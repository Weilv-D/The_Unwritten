# 三玄谱系

> 古时三玄在一檐。
>
> 三玄主为古师座下弟子，同门悟道，各立传承。每一玄皆有辈分诗，四字一序，世代相承。首字为第一代弟子，依次而下。
>
> 三玄主皆已前往天外。所留传承，散于天下。亲疏远近、庞杂交织——天下求道者，皆出三玄。

---

## 一、三玄主

古师之弟子。师名无考——被系统性地从所有典籍中抹去了。

三玄主为同门，悟道各异。在古师门下所修皆为服气养性道，后因道见不同，各立门户，各定辈分诗。

| 称号 | 辈分诗 | 悟道 | 去向 |
|------|--------|------|------|
| 通玄主 | 道藏希微 | 性 | 天外 |
| 兜玄主 | 元太洞衍 | 命 | 天外 |
| 青玄主 | 长策执玄 | 生 | 天外 |

三玄主最终皆前往天外。去时留下完整传承——弟子、果位、洞天、辈分诗、道统。

一代弟子亦皆证仙君（道胎圆满），其后陆续前往天外。去时各有安排——或传果位于后人，或将洞天托付显世道统。各人所证果位，有灿然载于典册者，亦有隐而不彰、后世只以"不详"或"不证果"记之者——然非无果位，只是不为世间所知。世间典籍对诸仙君之去向记载混乱：或曰飞升，或曰坐化，或曰不知所终——唯直系传承方知其详。

> 三玄主与一代弟子前往天外，是全书最大之背景悬念。他们去天外做什么？还会回来吗？卷五线索。

---

## 二、辈分诗与传承规制

三玄各定四字辈分诗，代代相传：

| 玄 | 辈分诗 | 第一代 | 第二代 | 第三代 | 第四代 |
|---|--------|-------|-------|-------|-------|
| 通玄 | 道藏希微 | **道** | 藏 | 希 | 微 |
| 兜玄 | 元太洞衍 | **元** | 太 | 洞 | 衍 |
| 青玄 | 长策执玄 | **长** | 策 | 执 | **玄** |

弟子名取辈分字加一字为号。如通玄第一代弟子"道乙"、兜玄第一代弟子"元曦"、青玄第一代弟子"长庚"。

每位弟子之传承即洞天。据果位、辟洞天、立显世道统。后世所见之宗门、世家、天坛，皆为洞天显世之面目。

> 位格极高者，典籍中只留名号与果位，不知其详。如太阳果位之主，世人不以辈分名称之，只尊号**东君**。

## 三、通玄谱系

> 辈分诗：**道藏希微**。当世通玄弟子多为第一代"道"字辈及第二代"藏"字辈。
>
> 通玄一代弟子六人，皆证仙君，陆续前往天外。

```
古师
  └─ 通玄主（前往天外）
       ├─ 道乙 ─→【离火果位】→ 朱明曜真天
       │    └─ 紫霄宗（显世道统）
       │
       ├─ 道虚 ─→【守藏果位】（史载"不证果"）
       │    └─ 藏真一脉（是否存续至今存疑）
       │
       ├─ 道微 ─→【少阳果位】→ 太微正阳天
       │    └─ 太玄门（显世道统·存疑）
       │
       ├─ 道都 ─→【坎水果位】→ 太冥坎元天
       │    └─ 青冥院（显世道统·存疑）
       │
       ├─ 道岚 ─→【果位不详】（不辟洞天，行走天下）
       │    └─ 散修精神源头
       │
       └─ 道漪 ─→【果位不详】（通玄修史传统之始）
            └─ 江陵藏书楼（精神源头之一）
```

### 道乙

离火果位。开辟朱明曜真天。证仙君后前往天外。后世尊为"离火之祖"。

道乙乃通玄主座下最先证果之弟子，亦是三玄弟子中最先开辟洞天之人。朱明曜真天坐落崑嵐山深处，为天下最古老之洞天之一。道乙前往天外后，将离火果位与洞天传于后人。显世道统为**紫霄宗**——筑基掌门、紫府长老，背后朱明曜真天。

### 道虚

**守藏果位（史载"不证果"）。未辟洞天。**

所有正史皆不载此人，唯有通玄秘典夹注中有一句："道虚受命守藏，不入世、不传人、不证果。"

正史所谓"不证果"，实乃道虚所证之果位不在世间常规认知之中。守藏者，守护服气养性道之核心残篇——此即其道、其果。不辟洞天，因其道不在开辟，而在保存。若这些残篇仍在，道虚一脉或以某种极其隐秘之方式存续至今。

> 剧情钩子：紫霄宗声称藏有上古残篇——是否与道虚一脉有关？

### 道微

少阳果位。开辟太微正阳天。证仙君后前往天外。

道微性烈，以斗法破境。将少阳传承留于显世。后世太玄门自称"道微传人"——以武立门之门风与道微一脉相合。太玄门于魏朝修史之祸中扮演了争议极大之角色，有人认为他们篡改了自家谱系。

### 道都

坎水果位。开辟太冥坎元天。证仙君后前往天外。

道都不擅斗法，专精阵法。将通玄主性功理论与荆扬巫觋阵法融而为一，开创"以性驭阵"之传统。后世青冥院是否为道都传人？院方既不承认亦不否认。

### 道岚

**果位不详。不辟洞天。** 证仙君后前往天外。

道岚行走天下，见缘收徒。其所证果位为何，典籍无载——唯有一句常被散修引用："天外有天，何须洞天？" 后世散修将道岚视为精神源头。

### 道漪

**果位不详。** 证仙君后前往天外。

通玄修史传统之始。道漪之使命乃记录修行界一切事件——公正地、完整地。此公正传统始于道漪，终于魏朝修史之祸中被彻底败坏。江陵藏书楼地库中最古老之那批通玄典籍，据说出自道漪门下。

## 四、兜玄谱系

> 辈分诗：**元太洞衍**。当世兜玄弟子多为第一代"元"字辈之传承分支。
>
> 兜玄一代弟子六人，皆证仙君，陆续前往天外。

```
古师
  └─ 兜玄主（前往天外）
       ├─ 元曦 ─→【明阳果位】→ 东华昭明天
       │    └─ 天坛祭祀体系之辅
       │
       ├─ 元霆 ─→【玄雷果位】→ 太乙神刑天
       │    └─ 玄雷宫（执天罚、司天纲千年）
       │         ├─ 历代玄雷果位持有者
       │         │    └─ 第一玄雷仙君·神雷（离职求道）
       │         └─ 雷宫覆灭，太乙神刑天失落
       │              └─ 庭尧（故尊复现）→ 真魂 → 江既白
       │
       ├─ 元祀 ─→【厥阴果位】→ 幽都玄祀天
       │    └─ 天坛祭祀体系之始 → 玄都观
       │
       ├─ 元朔 ─→【广莫果位】→ 幽朔玄冬天
       │    └─ 边镇一脉 → 沉氏（存疑）
       │
       ├─ 元玑 ─→【果位不详】
       │    └─ 商脉与情报网络之始
       │
       └─ 元渊 ─→【果位不详】（存在与否，争议极大）
            └─ 兜玄暗线之始
```

### 元曦

**明阳果位。开辟东华昭明天。证仙君后前往天外。**

元曦乃兜玄主座下大弟子。明阳果位为阴阳道统之一——明者昭明，阳者正阳，佐成天地之光明。元曦以此果位开辟东华昭明天，辅天坛祭祀体系之运行。显世传承散于后世，已不可详考。

> 后世记载混乱：有说元曦持太阳果位，有说明阳。混淆之因或在于"明阳"与"太阳"二字易混，亦或后人有意为之以抬高元曦之位格。

### 元霆（玄雷之主）

**玄雷果位。开辟太乙神刑天。证仙君后前往天外。**

元霆证玄雷果位后立玄雷宫，执天罚、司天纲，为天道秩序之刃。雷宫存续千年，历代果位持有者中最杰出者为**神雷**（第一玄雷仙君），道胎圆满之境，后离职求道，去向不明。

雷宫覆灭后，太乙神刑天成为失落洞天。五百年前故尊**庭尧**复现——然旋即身亡。太阴之主将其真魂送入轮回，混入陈恪魂魄，化为江既白。

### 元祀

**厥阴果位。开辟幽都玄祀天。证仙君后前往天外。**

元祀乃兜玄弟子中最沉默之一位。其贡献在于**制度**：将兜玄主"调诸祭、事正人"之理念化为天坛祭祀体系，创立玄都观为总坛。厥阴果位之权柄与祭祀、生死边缘之地相关。元祀以此权柄构建了天下最庞大之修行组织网络。传承留于显世。

后世天衡世家自称"元祀传人"——中间隔了多少代、多少次谱系断层，谁也说不清。

### 元朔

**广莫果位。开辟幽朔玄冬天。证仙君后前往天外。**

广莫为八风之一，主北方之风。元朔以广莫果位镇守北境，开辟幽朔玄冬天于燕代群山之中。

后世沉氏是否为元朔传人？沉氏讳莫如深。然有二端线索：祭祀传统带玄雷色彩（与元霆一脉之渊源），军镇编制与幽朔玄冬天规制高度相似。最可能之解释：雷宫覆灭后，元霆残余与元朔一脉合流——沉氏乃两脉融合之产物，以幽朔玄冬天为根基，以玄雷遗志为暗脉。

> 剧情钩子：卷一核心。沉氏"守墓人"守护的是什么？幽朔玄冬天是否仍在运转？

### 元玑

**果位不详。** 证仙君后前往天外。

典籍记载极少，只有一句反复出现："善通货殖，通达四方。"后世商脉与情报网络是否源于元玑？商人们乐意攀附，学者将信将疑。

### 元渊

**果位不详。** 证仙君后前往天外。存在与否，争议极大。

几乎所有正史不载此人。唯有青玄口述传统中偶尔提到："兜玄主有一个弟子，谁也不知道他做了什么，但兜玄的敌人总是莫名其妙地出事。"

## 五、青玄谱系

> 辈分诗：**长策执玄**。当世青玄弟子多为第一代"长"字辈之传承分支。
>
> 青玄一代弟子七人，皆证仙君，陆续前往天外。

```
古师
  └─ 青玄主（前往天外）
       ├─ 长殷 ─→【太阴果位】→ 玄冥太渊天
       │    └─ 教化与祭祀传统
       │
       ├─ 长＿（东君） ─→【太阳果位】→ 扶桑九曜天
       │    └─ 世号：东君。梁朝后期前往天外，扶桑九曜天因失主而崩解。
       │         └─ 太阴沉默 → 三玄阋墙
       │
       ├─ 长胥 ─→【霄雷果位】→ 玄霄灭影天
       │    └─ 武行传统 → 落霞（精神源头之一）
       │
       ├─ 长庚 ─→【庚金果位】→ 太白铸化天
       │    └─ 济世传统（医者匠人网络）
       │
       ├─ 长渊 ─→【果位不详】
       │    └─ 隐脉传统（地下生存术）
       │
       ├─ 长歌 ─→【清明果位】→ 朗照无蔽天
       │    └─ 记录传统（知识对抗蒙昧）
       │
       └─ 长息 ─→【果位不详】（存在与否，最大争议）
            └─ 因果暗线
```

### 长殷

**太阴果位。开辟玄冥太渊天。证仙君后前往天外。**

长殷乃青玄主座下最先证果之弟子。太阴果位为阴阳道统之阴极——与太阳互为表里。长殷以太阴证果，开辟玄冥太渊天。与通玄和兜玄之大弟子不同，长殷未建庞大显世组织——他行遍天下，于每一城镇教授基础修行与知识。后世是否有长殷传人存续？无人敢断言。

> 太阴果位与太阴之主的关系是全书核心暗线之一。长殷前往天外后，太阴果位空悬——然世间以"太阴沉默"载之，非空悬，而更似某种主动的隐匿。

### 长＿（东君）

**太阳果位。开辟扶桑九曜天。证仙君后前往天外。**

位格极高。典籍中不以辈分名称之，只尊号**东君**——取自古祭日之辞，见楚辞九歌。辈分名待考。

东君在世之日，太阳果位昌明，阴阳佐成运转如常。梁朝后期，东君前往天外。扶桑九曜天因失主而崩解。

太阳之陨后，太阴按理当显世。然太阴沉默——一点动静皆无。此一异常遂引发三玄阋墙。

> 世间多以"身死道消"载东君之终——唯扶桑九曜天直系传承知其真实去向。然洞天已崩，直系散落，真相湮没于史书之涂改与焚毁中。
>
> 剧情钩子：东君前往天外是自主决定，还是被人引导？他的离去与太阴之主有没有关系？

### 长胥

**霄雷果位。开辟玄霄灭影天。** 证仙君后前往天外。

霄雷为三雷之一，属阴。玄雷乃天罚之雷——刚猛正大；霄雷乃暗夜之雷——阴冷迅疾，一击即杀。长胥以霄雷证果，走的是青玄弟子中唯一之武道。

后世落霞之"以力抗仙"传统与长胥精神高度契合——虽然落霞从不声称任何古师传承。

### 长庚

**庚金果位。开辟太白铸化天。** 证仙君后前往天外。

庚金为五行五德中金德之"革"位——变革之金。长庚以庚金证果，其权柄与"变革、破旧立新"相关。然他将此权柄用于锻造与医术——为天下苍生制造工具、治病救人。后世散布天下之医者、匠人网络，据说始于长庚传承。

### 长渊

**果位不详。** 证仙君后前往天外。

青玄口述传统中对长渊之记载最为矛盾——有人说是最弱弟子，有人说是最强弟子，有人说"长渊不存在，只是青玄主的影子"。唯一一致之描述：长渊教会了青玄修士如何在不可能之境遇中生存。

### 长歌

**清明果位。开辟朗照无蔽天。** 证仙君后前往天外。

清明为八风之一。长歌以清明果位证道，权柄与"清明、透彻、无蔽"相关。长歌乃青玄弟子中之"记录者"——不同于通玄道漪记录修行界大事，长歌记录的是**凡人之事**。江陵藏书楼地库中最古老之那批记录，据说出自长歌门下。

### 长息

**果位不详。** 证仙君后前往天外。存在与否，最大争议。

"青玄主有一个弟子，他什么都不做，只是在合适的时候出现在合适的地方。"若长息确实存在，其修行或是**改变因果**——不在蛮力改变世界，而在关键节点轻轻推一把。此种修行方式与太阴之主之布局行为有惊人相似。

> 长息、元渊、太阴之主——三者皆擅长"暗中行事"。若三者有关联，三玄谱系之下或还有一条更深之暗脉。卷五揭示。

---

## 六、跨谱系要者

### 周饶

梁朝史官。身系三玄，书通六家。提出"三德说"与"阴阳佐成"之论。师承不明——一个"你中有我"之活例。

### 崔谕 / 李道枰

存疑派代表人物。分别著【谬古论】与【无史说】。师承皆不明。他们接触到之独家材料，由谁提供？

### 二吕

连名字都统一不了。他们"证而不得"——证出来了，然天地不认。或许他们试图走三玄未分之前的老路，但天地已变，老路走不通。

---

## 七、谱系暗线

### 太阴之主

太阴之主不属于三玄任何一脉。然太阴乃阴阳道统之一部分——太阳、太阴同归青玄一门，分别由东君与长殷所持。东君前往天外后，太阴沉默。

太阴之主是谁？是长殷一脉之隐秘延续，还是天地法则本身所生之存在？元渊、长息、太阴之主——三者之间的关系，乃全书最后之谜团。

### 薛霖卿

末代通玄宫宫主。某夜与不明身份者密谈后，亲手解散通玄宫。通玄宫乃通玄主亲自建立之传承中枢——薛霖卿为何要解散它？那个密谈者是谁？

### 神雷

第一玄雷仙君，道胎圆满之境。离职求道，去向不明。他所求之"道"是什么？与三玄主前往天外是否同一件事？
