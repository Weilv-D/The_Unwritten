# 三玄谱系

> 古时三玄在一檐。
>
> 三玄主为古师座下弟子，同门悟道，各立传承。每一玄皆有辈分诗，四字一序，世代相承。首字为第一代弟子，依次而下。
>
> 三玄主皆已前往天外。所留传承，散于天下。亲疏远近、庞杂交织——天下求道者，皆出三玄。

---

## 一、三玄主

古师之弟子。师名无考——被系统性地从所有典籍中抹去了。

三玄主为同门，悟道各异。在古师门下所修皆为服气养性道，后因道见不同，各立门户，各定辈分诗。

| 称号 | 辈分诗 | 悟道 | 去向 |
|------|--------|------|------|
| 通玄主 | 清都玉京 | 性 | 天外 |
| 兜玄主 | 承天正序 | 命 | 天外 |
| 青玄主 | 长策执玄 | 生 | 天外 |

三玄主最终皆前往天外。去时留下完整传承——弟子、果位、洞天、辈分诗、道统。

> 三玄主前往天外是全书最大的背景悬念。他们去天外做什么？还会回来吗？卷五线索。

---

## 二、辈分诗与传承规制

三玄各定四字辈分诗，代代相传：

| 玄 | 辈分诗 | 第一代 | 第二代 | 第三代 | 第四代 |
|---|--------|-------|-------|-------|-------|
| 通玄 | 清都玉京 | **清** | 都 | 玉 | 京 |
| 兜玄 | 承天正序 | **承** | 天 | 正 | 序 |
| 青玄 | 长策执玄 | **长** | 策 | 执 | **玄** |

弟子名取辈分字加一字为号。如通玄第一代弟子"清乙"、兜玄第一代弟子"承曦"、青玄第一代弟子"长庚"。

每位弟子的传承即洞天。据果位、辟洞天、立显世道统。后世所见之宗门、世家、天坛，皆为洞天显世之面目。

> 位格极高者，典籍中只留名号与果位，不知其详。如太阳果位之主，世人不以辈分名称之，只尊号**东君**。

---

## 三、通玄谱系

> 辈分诗：**清都玉京**。当前天下通玄弟子多为第一代"清"字辈及第二代"都"字辈。

```
古师
  └─ 通玄主（前往天外）
       ├─ 清乙 ─→【离火果位】→ 离火洞天
       │    └─ 紫霄宗（显世道统）
       │
       ├─ 清虚 ─→ 守护上古服气养性道核心残篇（隐脉）
       │    └─ 藏真一脉（是否存续至今存疑）
       │
       ├─ 清微 ─→【少阳果位】→ 少阳洞天
       │    └─ 太玄门（显世道统·存疑）
       │
       ├─ 清都 ─→【坎水果位】→ 坎水洞天
       │    └─ 青冥院（显世道统·存疑）
       │
       ├─ 清岚 ─→ 散修精神源头（不辟洞天）
       │
       └─ 清漪 ─→ 通玄修史传统之始
```

### 清乙

离火果位。开辟离火洞天。后世尊为"离火之祖"。

清乙是通玄主座下最先证果的弟子，也是三玄弟子中最先开辟洞天的人。离火洞天坐落崑嵐山深处，为天下最古老的真君洞天之一。显世道统为**紫霄宗**——筑基掌门、紫府长老、背后离火洞天。

### 清虚

**果位不详。洞天不详。**

所有正史都不载此人，唯有通玄秘典夹注中有一句："清虚受命守藏，不入世、不传人、不证果。"

不证果——意味着清虚从未开辟洞天。但他的使命是守护服气养性道的核心残篇。如果这些残篇仍在，清虚一脉可能以某种极其隐秘的方式存续至今。

> 剧情钩子：紫霄宗声称藏有上古残篇——是否与清虚一脉有关？

### 清微

少阳果位。开辟少阳洞天。

清微性烈，以斗法破境。后世太玄门自称"清微传人"——以武立门的门风与清微一脉相合。太玄门在魏朝修史之祸中扮演了争议极大的角色，有人认为他们篡改了自己的谱系。

### 清都

坎水果位。开辟坎水洞天。

清都不擅斗法，专精阵法。将通玄主性功理论与荆扬巫觋阵法结合，开创"以性驭阵"传统。后世青冥院是否为清都传人？院方既不承认也不否认。

### 清岚

不证果，不辟洞天。行走天下，见缘收徒。

"天外有天，何须洞天？"后世散修将清岚视为精神源头。

### 清漪

通玄修史传统之始。清漪的使命是记录修行界一切事件——公正地、完整地。这一公正传统始于清漪，终于魏朝修史之祸中被彻底败坏。

## 四、兜玄谱系

> 辈分诗：**承天正序**。当前天下兜玄弟子多为第一代"承"字辈的传承分支。

```
古师
  └─ 兜玄主（前往天外）
       ├─ 承曦 ─→【太阳果位】→ 太阳洞天
       │    └─ 世号：东君。梁朝后期身死道消，太阳洞天崩解。
       │         └─ 太阴沉默 → 三玄阋墙
       │
       ├─ 承霆 ─→【玄雷果位】→ 玄雷洞天
       │    └─ 玄雷宫（执天罚、司天纲千年）
       │         ├─ 历代玄雷果位持有者
       │         │    └─ 第一玄雷仙君·神雷（离职求道）
       │         └─ 雷宫覆灭，玄雷洞天失落
       │              └─ 庭尧（故尊复现）→ 真魂 → 江既白
       │
       ├─ 承祀 ─→【厥阴果位】→ 厥阴洞天
       │    └─ 天坛祭祀体系之始 → 玄都观
       │
       ├─ 承朔 ─→【广莫果位】→ 广莫洞天
       │    └─ 边镇一脉 → 沉氏（存疑）
       │
       ├─ 承玑 ─→ 果位存疑·洞天不详
       │    └─ 商脉与情报网络之始
       │
       └─ 承渊 ─→ 果位不详·洞天不详·存在存疑
            └─ 兜玄暗线之始
```

### 承曦（东君）

**太阳果位。开辟太阳洞天。**

位格极高。典籍中不以辈分名称之，只尊号**东君**——取自古祭日之辞，见楚辞九歌。

东君在世时，太阳果位昌明，阴阳佐成运转如常。梁朝后期，东君尝试向道胎圆满突破——失败了。身死道消，太阳洞天崩解。

太阳之陨后，太阴按理当显世。但太阴沉默——一点动静都没有。这一异常引发了三玄阋墙。

> 东君只留下一个名号。他是谁、为何要突破道胎圆满、他的死与太阴沉默有无关联——这些答案随太阳洞天崩解而消散。
>
> 剧情钩子：东君突破道胎圆满是自主决定，还是被人引导？他的死与太阴之主有没有关系？

### 承霆（玄雷之主）

**玄雷果位。开辟玄雷洞天。**

承霆证玄雷果位后立玄雷宫，执天罚、司天纲，为天道秩序之刃。雷宫存续千年，历代果位持有者中最杰出者为**神雷**（第一玄雷仙君），道胎圆满之境，后离职求道，去向不明。

雷宫覆灭后，玄雷洞天成为失落洞天。五百年前故尊**庭尧**复现——但旋即身亡。太阴之主将其真魂送入轮回，混入陈恪魂魄，化为江既白。

### 承祀

**厥阴果位。开辟厥阴洞天。**

承祀是兜玄六弟子中最安静的一位。他的贡献是**制度**：将兜玄主"调诸祭、事正人"的理念转化为天坛祭祀体系，创立玄都观为总坛。厥阴果位的权柄与祭祀、生死边缘地带相关。承祀用这份权柄构建了天下最庞大的修行组织网络。

后世天衡世家自称"承祀传人"——中间隔了多少代、多少次谱系断层，谁也说不清。

### 承朔

**广莫果位。开辟广莫洞天。**

广莫为八风之一，主北方之风。承朔以广莫果位镇守北境，开辟广莫洞天于燕代群山之中。

后世沉氏是否为承朔传人？沉氏讳莫如深。但有两个线索：祭祀传统带玄雷色彩（与承霆一脉的渊源），军镇编制与广莫洞天规制高度相似。最可能的解释：雷宫覆灭后，承霆残余与承朔一脉合流——沉氏是两脉融合的产物，以广莫洞天为根基，以玄雷遗志为暗脉。

> 剧情钩子：卷一核心。沉氏"守墓人"守护的是什么？广莫洞天是否仍在运转？

### 承玑

**果位存疑。洞天不详。**

典籍记载极少，只有一句反复出现："善通货殖，通达四方。"后世商脉和情报网络是否源于承玑？商人们乐意攀附，学者将信将疑。

### 承渊

**果位不详。洞天不详。存在与否，争议极大。**

几乎所有正史不载此人。唯有青玄口述传统中偶尔提到："兜玄主有一个弟子，谁也不知道他做了什么，但兜玄的敌人总是莫名其妙地出事。"

## 五、青玄谱系

> 辈分诗：**长策执玄**。当前天下青玄弟子多为第一代"长"字辈的传承分支。

```
古师
  └─ 青玄主（前往天外）
       ├─ 长殷 ─→【明阳果位】→ 明阳洞天
       │    └─ 教化传统 → 明德书院（精神源头）
       │
       ├─ 长胥 ─→【霄雷果位】→ 霄雷洞天
       │    └─ 武行传统 → 落霞（精神源头之一）
       │
       ├─ 长庚 ─→【庚金果位】→ 庚金洞天
       │    └─ 济世传统（医者匠人网络）
       │
       ├─ 长渊 ─→ 果位不详·洞天不详
       │    └─ 隐脉传统（地下生存术）
       │
       ├─ 长歌 ─→【清明果位】→ 清明洞天
       │    └─ 记录传统（知识对抗蒙昧）
       │
       └─ 长息 ─→ 果位不详·洞天不详·存在存疑
            └─ 因果暗线
```

### 长殷

**明阳果位。开辟明阳洞天。**

长殷是青玄主座下最先证果的弟子。与通玄和兜玄的大弟子不同，长殷没有建立庞大的显世组织——他行遍天下，在每个城镇教授基础修行和知识。后世明德书院是否为长殷传人？书院既不承认也不否认。

### 长胥

**霄雷果位。开辟霄雷洞天。**

霄雷为三雷之一，属阴。玄雷是天罚之雷——刚猛正大；霄雷是暗夜之雷——阴冷迅疾，一击即杀。长胥以霄雷证果，走的是青玄弟子中唯一的武道。

后世落霞的"以力抗仙"传统与长胥精神高度契合——虽然落霞从不声称任何古师传承。

### 长庚

**庚金果位。开辟庚金洞天。**

庚金为五行五德中金德的"革"位——变革之金。长庚以庚金证果，其权柄与"变革、破旧立新"相关。但他将这份权柄用于锻造和医术——为天下苍生制造工具、治病救人。后世散布天下的医者、匠人网络，据说始于长庚传承。

### 长渊

**果位不详。洞天不详。**

青玄口述传统中对长渊的记载最为矛盾——有人说是最弱弟子，有人说是最强弟子，有人说"长渊不存在，只是青玄主的影子"。唯一一致的描述是：长渊教会了青玄修士如何在不可能的环境中生存。

### 长歌

**清明果位。开辟清明洞天。**

清明为八风之一。长歌以清明果位证道，权柄与"清明、透彻、无蔽"相关。长歌是青玄弟子中的"记录者"——不同于通玄清漪记录修行界大事，长歌记录的是**普通人的事**。江陵藏书楼地库中最古老的那批记录，据说出自长歌门下。

### 长息

**果位不详。洞天不详。存在与否，最大争议。**

"青玄主有一个弟子，他什么都不做，只是在合适的时候出现在合适的地方。"如果长息确实存在，他的修行可能是**改变因果**——不在蛮力改变世界，而在关键节点轻轻推一把。这种修行方式与太阴之主的布局行为有惊人的相似。

> 长息、承渊、太阴之主——三者都擅长"暗中行事"。如果三者有关联，三玄谱系之下可能还有一条更深的暗脉。卷五揭示。

## 六、跨谱系要者

### 周饶

梁朝史官。身系三玄，书通六家。提出"三德说"和"阴阳佐成"理论。师承不明——一个"你中有我"的活例子。

### 崔谕 / 李道枰

存疑派代表人物。分别著【谬古论】和【无史说】。师承都不明。他们接触到的独家材料，是谁提供的？

### 二吕

连名字都统一不了。他们"证而不得"——证出来了，但天地不认。也许他们试图走三玄未分之前的老路，但天地已变，老路走不通。

---

## 七、谱系暗线

### 太阴之主的位置

太阴之主不属于三玄任何一脉。但太阴是阴阳道统的一部分。太阳果位由兜玄承曦（东君）所持。东君死后，太阴沉默。

太阴之主是谁？是某个谱系的隐秘延续，还是天地法则本身生出的存在？承渊、长息、太阴之主——三者之间的关系，是全书最后的谜团之一。

### 薛霖卿

末代通玄宫宫主。某夜与不明身份者密谈后，亲手解散通玄宫。通玄宫是通玄主亲自建立的传承中枢——薛霖卿为何要解散它？那个密谈者是谁？

### 神雷

第一玄雷仙君，道胎圆满之境。离职求道，去向不明。他追求的是什么"道"？与三玄主前往天外是否同一件事？
