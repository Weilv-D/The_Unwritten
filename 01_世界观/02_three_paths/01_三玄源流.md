# 三玄源流

## 一、总论

三玄者，通玄、兜玄、青玄，皆出正始道人门下。三玄之法，同源而异流——非功法之异，乃对"德"之理解各有侧重。

正始道人以阴阳剖混沌，定五德，分十二炁，传三弟子。长曰通玄，次曰兜玄，季曰青玄。初时三玄弟子共居玉京，同祀正始，号"三玄共檐"。

三玄主未曾决裂——只是各走各路。弟子之间因嫌隙而生冲突，然三玄主始终不曾公开对立，三玄弟子亦从未明面决裂。三玄共檐之义，名义上从未断绝。

**三玄同修仙道，各有果位登临者。** 通玄以仙统自居，但兜玄、青玄亦不乏真君、大真人。天地业位体系（61道、305神通、果位闰余）为三玄共有之修行框架，不专属于任何一玄。三玄之分在于道统哲学之侧重，不在于修行方法或境界高下。

**三玄你中有我，我中有你。** 历史上，青玄子弟行兜玄之事、通玄子弟行青玄之事，屡见不鲜。弟子跨玄拜师、宗门传承混合多玄之法，是常态而非例外。将任何大事件简单归因于某一玄，都是后人约化之辞。

## 二、通玄

### 核心理念

侧重仙德——修士当超凡脱俗，以仙道为尊。通玄弟子善于体系化传承，仙宗制度与师承秩序，多由通玄一脉奠定。

### 关键史事

| 时期 | 事件 | 影响 |
|------|------|------|
| 太古末 | 仙神大战中为主要攻方之一 | 太古神圣陨落殆尽 |
| 上古中期 | 九真混成，立三十六仙宗 | 确立仙道秩序（但九真非纯通玄） |
| 上古末 | 卷入宫移羽盖之变 | 兜玄势力大衰 |
| 近古后期 | 参与三玄旧盟（但内部态度分裂） | 盟约维系百年后废 |

### 九真与大道对应

九真为首批证真君者，各开一脉。后世多将九真归为"通玄弟子"，此为修史之简化——九真之中有通玄弟子，亦有兜玄、青玄门下。其名号与天地业位之大道直接对应：

| 九真 | 所开一脉 | 当世状态 |
|------|---------|---------|
| 少阳真君 | 少阳 | 已陨落 |
| 太阴真君 | 太阴/月宫 | 已隐世 |
| 冲虚真君 | 玉虚 | 近古西迁，今在西方 |
| 紫虚真君 | 紫府 | — |
| 清虚真君 | 清都 | 三度废立终绝，余脉为太微玄宫 |
| 洞虚真君 | 洞华 | — |
| 三人名号佚 | — | — |

### 当世格局

通玄系仙宗数量最多、势力最盛，然各立山头，不复上古之团结。真君尚存数位。但"通玄系"仙宗中，不少已混合多玄传承，纯粹的通玄宗门反而在少数。

## 三、兜玄

### 核心理念

侧重神德——天地万灵各有其主，修士当敬神驭鬼。北极驱邪院即此理念之最高实践——总摄天下鬼神，祀灵坛祀太古战死之神。

兜玄弟子同样修仙道、证果位。北极驱邪院中历代院主多为真君境界。兜玄与通玄的区别不在于"修不修仙"，而在于对"仙"之定位——兜玄以为仙之上尚有神，修仙只是手段，敬神驭鬼才是目的。

### 关键史事

| 时期 | 事件 | 影响 |
|------|------|------|
| 太古末 | 仙神大战中为主守方之一 | 大败，太古神圣陨落殆尽 |
| 上古初 | 立北极驱邪院（初建时三玄共参） | 兜玄逐渐独占该院 |
| 上古末 | 宫移羽盖之变 | 封院三百年，势力大衰 |
| 中古中期 | 观复道人收集古道藏（观复横跨三玄人脉） | 事败 |
| 近古后期 | 参与三玄旧盟（内部态度分裂） | — |

### 当世格局

北极驱邪院封院已久。但兜玄弟子散布于各仙宗之中，并非真的"不涉世事"——不少仙宗的护法、祭祀体系中，都有兜玄传承的影子。历代院主以秘法传承，该院从未真正断绝。

## 四、青玄

### 核心理念

侧重人德——唯有人道为根本。青玄之"人德"，非贬低修士，而是主张修行之终极目的应是"人"之圆满。

青玄弟子同样修仙道、证果位。青玄与通玄、兜玄的区别不在于修行路径，而在于价值取向。正始之道之所以门槛极高、斗法孱弱，正因其核心理念（性命同修，不假外物）与青玄人德一脉相承——但正始之道不是青玄的专属修行方法，通玄、兜玄弟子中亦有人修习正始之道的残篇。

### 关键史事

| 时期 | 事件 | 影响 |
|------|------|------|
| 太古末 | 仙神之争中周旋 | 保存实力最多 |
| 上古初 | 授龙蹻化龙之法（青玄主代师收徒） | 龙属一脉辈份极高 |
| 上古末 | 卷入宫移羽盖之变（参与程度有争议） | — |
| 中古初 | 龙属北迁有青玄主授意之成分（但亦有龙属自主意志） | 龙属入主东海 |
| 中古 | 清都观三度废立（通玄、兜玄皆有参与） | 清都终废 |
| 近古 | 清都余脉参与太微玄宫之立（但非唯一主导力量） | 暗中影响东方格局 |

### 青玄之法的渗透

清都观虽废，青玄之法从未消失。其核心理念已渗入各宗：
- 各宗门中"以心传心"的密传，多带青玄色彩
- 凡人国度的"仁政"理念，与青玄人德有呼应
- 但这些渗透是自然流变的结果，而非青玄主的刻意安排

### 当世格局

青玄再无核心宗门。弟子散落各方，或隐修，或依附他宗。龙属为青玄最明确的传承分支，据东海。但龙属作为独立势力，自有其利益考量，不完全听从青玄主调遣。

## 五、三玄主与天外

三玄主（通玄主、兜玄主、青玄主）于太古末已前往天外，不在现世。古代仙君亦大都前往天外求道。天外者，主界壁障之外也——非洞天（洞天仍在壁障之内），而是彻底脱离主界的存在。

三玄主临行前是否留下法旨或权柄之锚？各玄真君口传不一，无从确证。

> 三玄主前往天外之后，主界之修行秩序实际上由果位持有者（真君、仙人）自行维系。三玄主之"旨意"多为各玄后人依自身立场诠释，是否真有三玄主之法旨存世，已成悬案。

## 六、三玄之交错

三玄之分，是哲学倾向之分，不是阵营之分。以下为历史上三玄交错的典型例子：

| 事件 | 交错表现 |
|------|---------|
| 九真混成 | 九真跨越三玄，非通玄专属 |
| 北极驱邪院初建 | 三玄共参，后来兜玄独占 |
| 仙神大战 | 三玄弟子并非整齐对立，有倒戈、骑墙、暗中通敌 |
| 宫移羽盖之变 | 参与者成分不明，可能涉及三玄之外的第三方力量 |
| 清都观废立 | 通玄、兜玄弟子皆有参与，非青玄内部事务 |
| 六天故气之乱 | 信众来自各玄，非某一玄的叛逆 |
| 大晟之世 | 兜玄主导，废三千二百宗，立昭元一府，但他玄弟子亦有任职 |
| 从革之变 | 阵营随时变动，三玄之别几乎失去意义 |
| 三玄旧盟 | 支持与反对者跨三玄分布，非按玄站队 |
| 当世仙宗 | 多数混合多玄传承，弟子往往不自知 |

三德（仙德、神德、人德）为后人概括之标签。实际历史中，三玄弟子之行为远比"三德对应"复杂。将历史简化为三玄角力、三德之争，是修史者为方便记述而加之后见之明。
