# 势力总览

> 势力流变谱系（辈分体系、一代弟子总表、四君体系、世家流变）详见 `02_势力流变谱系.md`。本文件为各势力之当世状态概览。

## 一、当世格局概览

### 势力分级

| 级 | 类型 | 战力标准 | 外部形态 |
|----|------|---------|---------|
| 一 | 三玄道统 | 三玄主（道胎及以上） | 不以势力形态现世，为天下道统之源 |
| 二 | 道胎势力 | 道胎坐镇 | 如北极驱邪院曾经是 |
| 三 | 仙朝 · 仙宗 | 多位金丹 | 据半个天下以上（仙朝），或跨地域影响（仙宗） |
| 四 | 仙族 · 割据国 · 宗 | 一位金丹 | 仙族隐于洞天、割据国统治一方、宗传法收徒——同一种力量的三种面目，取决于真君之需求 |
| 五 | 门 · 世家 | 最高紫府 | 门为紫府级宗门（不曾出过金丹）；世家为祖上出过金丹但目前无金丹坐镇之家族 |

**关键区分——宗与门**

- **宗**：背后有金丹真君。外部代理可能是割据国，可能是宗门本身，也可能是隐于洞天的仙族——取决于真君的需求。割据国一般会统治境内的门，但宗一般不会。
- **门**：最高战力止于紫府。不曾出过金丹，是修行世界的基层组织。

**仙族与世家的区别**

- **仙族**（四级）：当前有金丹坐镇，与仙族/割据国/宗同级
- **世家**（五级）：祖上出过金丹但目前无金丹，最高战力止于紫府，与门同级——但世家的历史声望与人脉远超一般门

### 当世格局

三玄旧盟废后，天下复乱。仙宗相互攻伐，世家朝秦暮楚。此乱持续数百年，至当世犹未止。三玄主未曾决裂，弟子之争亦不曾明面撕破——三玄共檐之义，名义上从未断绝。

> 三玄你中有我、我中有你。当世多数仙宗混合了多玄传承，弟子往往不自知。以下按主要倾向归类，但不可理解为各势力"属于"某一玄。

当世主要势力：

| 势力 | 势力级别 | 主要传承倾向 | 地域 |
|------|---------|-------------|------|
| 北极驱邪院 | 二级（道胎势力，已封） | 兜玄为主 | 不明 |
| 各大宗门（存者不足十家） | 三级（仙宗） | 多玄混合 | 各地 |
| 云锦山 | 四级（宗） | 兜玄·神霄道统 | 荆扬 |
| 太微玄宫 | 四级（宗） | 玉虚+清都+多玄 | 东方 |
| 龙属 | 四级（宗） | 青玄渊源 | 东海 |
| 释修 | 四级（宗，外来） | 渊源不明 | 北方 |
| 十余古世家 | 五级（世家） | 各有依附 | 各地 |

## 二、通玄系·仙宗

### 沿革

上古中期，通玄弟子各立宗门，初时数百，经千年兼并定为三十六仙宗。天坠毁去近半，天下从革中再减，今存者不足十家。

三十六仙宗非定数，仅言其多。后世以为定数者，实为讹传。

### 希字辈后裔脉络

首批通玄·希字辈真君，名号今存其六。详见 `02_势力流变谱系.md`。

| 真君 | 成就 | 脉络演变 | 当世状态 |
|------|------|---------|---------|
| 希阳真君 | 真君 | 开少阳脉 | 已陨落 |
| 希暄真君 | 真君 | 开广寒脉 | 已隐世 |
| 希玄真君 | 仙君 → 天外 | 开玉虚脉 | 近古分裂：主脉西迁；东方余脉为太微玄宫 |
| 希辰真君 | 真君 | 开天枢脉 | — |
| 希清真君 | 仙君 → 天外 | 开清都脉（青玄主脉） | 三度废立终绝。余脉借壳为太微玄宫 |
| 希灵真君 | 真君 | 开洞华脉 | — |

### 仙宗内部结构（通则）

仙宗虽各有不同，然组织规制大同小异：
- **掌门**：真君或大真人，总领宗务
- **长老会**：紫府修士，参与决策
- **弟子**：筑基以上为骨干，练气为主力
- **附属势力**：矿脉、灵田、辖区

### 与世家的关系

仙宗立宗传法，世家以血脉传道。二者相互依存又相互提防：
- 仙宗需要世家提供灵脉、矿藏、凡人基础
- 世家需要仙宗提供功法传承、护道之力
- 世家背后往往有仙宗支持，仙宗背后亦有世家资助
- 大晟之世即为世家间接统治之典型——晟朝实为兜玄系仙朝

## 三、兜玄系·北极驱邪院

### 基本信息

| 项目 | 内容 |
|------|------|
| 别称 | 北院、驱邪院、神庭；释修称"帝释天"（讹传） |
| 性质 | 神庭——非宗门，类似天庭。北宫神雷，燮理天下，总摄鬼神 |
| 归属 | 兜玄祖庭 |
| 创立 | 上古初，仙神之争中 |
| 初代院主 | 执衡真君 |
| 核心建筑 | 祀灵坛（祀太古战死之神） |

### 历史

上古初立院，总摄天下鬼神。上古末宫移羽盖之变后封院三百年。中古派观复道人收集古道藏，事败。此后持续衰落，当世封院已久。

### 当世状态

- **公开认知**：已封院，院址不明，祀灵坛或已荒废
- **隐秘真相**：实未封。历代院主以秘法传承，在院中修行。祀灵坛中太古之神真灵尚存，不显于世
- **院中密室**：藏有正始观师像（自玉京山移来），历代院主口传

## 四、兜玄系·云锦山

### 基本信息

| 项目 | 内容 |
|------|------|
| 别称 | 南庭、雷法南庭、云锦天师府 |
| 性质 | 兜玄雷法祖庭之南庭 |
| 归属 | 兜玄·神霄道统 |
| 地域 | 荆扬 |
| 洞天 | 云锦天 |
| 势力层级 | 棋手级（有真君坐镇） |

### 沿革

兜玄雷法祖庭分南北二庭：北庭即北极驱邪院，南庭即云锦山。上古大字辈时期受命立南庭，初为北庭之附庸。宫移羽盖之变后北庭封院，南庭自立。中古辟云锦天，大晟时达到鼎盛。从革中经历传承之争，折衷后延续至今。

### 天师制度

"天师"为雷法祖庭之特殊尊称，始自驱邪院任命。一般为紫府大真人，负责驱邪镇煞、度人传箓。传承有血脉与择贤二说并行，历代争议不断。山中另有一位真君闭关，天师主外、真君居内。

### 与沈氏之关系

同为兜玄系在荆扬。沈氏管神祠祭祀，云锦山管驱邪镇煞，职能互补。大晟时沈氏为天师下属，从革中短暂对立，当世表面合作暗中竞争。

详细设定见 `03_云锦山.md`。

## 五、青玄系·太微玄宫

### 基本信息

| 项目 | 内容 |
|------|------|
| 别称 | 东玉虚、太微宗；玉虚主脉呼为"伪宫" |
| 归属 | 表面通玄（自称承玉虚正统），实则清微道统（清都余脉借壳） |
| 创立 | 近古中期 |
| 地域 | 东方 |

### 真实渊源

玉虚西狩后，东方余下的玉虚弟子联合部分世家立太微玄宫。表面自称承冲虚真君，实为清都观余脉借壳重生——功法表面玉虚一系，内核多有清都痕迹。

太微立后百年，内部"楼观宗圣之辨"：
- **楼观派**（主传法弘道，广收门徒）= 清都余脉 → 胜
- **宗圣派**（主隐修求道，不涉尘世）= 玉虚旧部 → 败

楼观派胜后清都余脉彻底掌控太微，势力大张，弟子出走别立宗门或入凡建世家，近古中后期仙宗格局多源于此。

### 内部分化

太微弟子自以为玉虚正宗，不知是清都余脉。此秘密仅极少数高层知晓。

### 基本信息

| 项目 | 内容 |
|------|------|
| 始祖 | 龙蹻【别：蹻龙、龙祖；龙属呼"老祖公"】 |

| 地域 | 东海（中古自南海北迁） |
| 势力层级 | 棋手级，当世最大势力之一 |

### 沿革

| 时期 | 事件 |
|------|------|
| 上古初 | 龙蹻入青玄门下，受化龙之法，开龙属一脉 |
| 上古 | 青玄主令龙属自立，不涉三玄内斗 |

### 特点

- 辈份极高：历代真君见龙蹻亦礼敬（代师收徒，相当于正始道人徒孙辈）
- 中立立场：名义上不涉三玄内斗，实则暗承青玄意旨
- 东海之主：原居东海之妖族或降或走

## 七、独立势力·释修

### 基本信息

天坠后自西方传来，入主北方，为当世新兴势力。不属三玄体系，修行路径与三玄正传迥异。

### 当世地位

- 在北方有实质影响力
- 与兜玄有教义冲突（称北极驱邪院为"帝释天"）
- 对三玄内争持观望态度
- 影响了近古以降的名号体系（"天尊"等称呼受释修影响）

---

## 八、世家

七十二世家为上古不愿立宗者，以血脉传道。此数目非实数，仅言其多。代有兴替，今世所谓"古世家"者不过十余姓。

世家特征：
- 以血脉传承功法，不公开收徒
- 各有依附的仙宗，亦被仙宗所依赖
- 部分世家入凡立国，大晟即其典型（然大晟实为兜玄主导，非纯世家）
- 大晟立昭元一府（居晟煌天），为有晟一朝的道统之法府，号"不为谁家道统"
- 晟朝亡后，昭元一府状态不明，从革中世家格局大洗牌

---

## 九、凡人

天下无凡人王朝。历朝历代皆仙朝——由修士直接统治，凡人为治下之民。

凡人无独立政权，聚居于修行势力辖区之内。仙宗辖地、世家领地、仙朝故土，皆有凡人耕作生息。凡人不知仙宗之名，不知三玄之别。只知山上有仙人、北方有和尚、东海有龙。

凡人于修士而言，是资粮来源（灵脉感应、矿藏开采、人口基数）与势力根基。大晟仙朝即为典范——以兜玄为根底，统一修行秩序，废三千二百宗，天下凡人尽入仙朝版籍。
