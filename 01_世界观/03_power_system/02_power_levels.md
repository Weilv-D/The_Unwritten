## 一、境界总纲

| 境界 | 尊称       | 核心特征                       | 寿元（约）       |
| :--- | :--------- | :----------------------------- | :--------------- |
| 胎息 | —         | 感应先天一炁，唤醒身内宇宙     | 七十余岁         |
| 练气 | —         | 引气入体，贯通十二重楼         | 一百五十余岁     |
| 筑基 | —         | 道基铸成，三宝初凝             | 三百余岁         |
| 紫府 | 真人       | 开辟泥丸宫，凝聚真魂，显化神通 | 八百至一千二百载 |
| 金丹 | 真君、诸君 | 熔炼金性，求得果位             | 与天同寿         |
| 道胎 | 仙人       | 金丹化道胎，超脱果位束缚       | 与天同寿         |

## 二、修行流程

凡人
  ↓ 灵窍觉醒
胎息 → 感应先天一炁，唤醒身内宇宙
  ↓ 引气入体
练气 → 积累法力，拓宽经脉
  ↓ 铸就道基
筑基 → 下丹田凝固道基，性命初定
  ↓ 抬升道基（逆流十二重楼）
紫府 → 道基入升阳府，显化神通，真魂凝聚
  ↓ 五神通圆满，熔炼金性
金丹 → 以金性感应果位，铸就不朽金丹，性命圆融
  ↓ 金丹化道胎
道胎 → 未知

## 三、修行时间参考

| 阶段                   |    平均耗时    | 备注                 |
| :--------------------- | :------------: | :------------------- |
| 胎息 → 练气           |     约5年     | 贯通六轮             |
| 练气 → 筑基           |     约30年     | 九层积累             |
| 筑基 → 紫府（一神通） |     约30年     | 首道生死关           |
| 紫府内每修成一神通     | 十数年至数十年 | 「参紫仙槛」可困百年 |
| 大真人 → 金丹         |      不定      | 或终身无望           |
