# 修行体系总纲

> 此卷论修行之玄机本源，为万法之宗、诸术之祖，凡涉修行种种设定，皆以此为准的。

---

## 一、性命二元论

### 总纲

修行者，以己之「性」参悟天地，交感万物，终而承负天地定位之「命」也。

### 性（内求之本）

- **所含**：法躯、修为、道行
- **所自**：凡由**内炼**所得、归乎**本我**之力皆是
- **所系**：直系**身寿**。性强则法躯坚凝，长生久视
- **其本**：「我」之力也

### 命（外应之凭）

- **所含**：位格、命数、魂魄
- **所自**：凡由**外感**所得、归乎**天地定位**之权柄皆是
- **所系**：直系**命寿**。命贵则位格崇隆，劫厄不加
- **其本**：「天地」赋乎「我」之权也

### 性命关系

- 性无命不立，命无性不灵
- 修行即性命双修，缺一不可
- 紫府境：性命初融，真魂始凝
- 金丹境：性命合一为不朽金性，求得果位，天地同寿

## 四、四大修行体系

> **三玄与四道**：三玄（通玄、兜玄、青玄）是立场与阵营，决定"你信什么"；四大修行体系是方法与门径，决定"你怎么修"。二者非一一对应——同属一玄者可修不同之道，然主流倾向有别：通玄多修紫府金丹道，兜玄多修青冥登天道，青玄多修黄庭内景道。

### 1. 紫府金丹道

- **要旨**：以性求命，内圣外王
- **门径**：先炼性功至金丹圆满，再以金性求取果位
- **次第**：
  - 筑基期铸造**道基**（如「君兑隅道基」）
  - 紫府期将道基抬入升阳府，显化为**神通**
  - 五神通圆满后熔炼为**不朽金性**，感应果位，成就金丹
- **神通与果位**：每一道神通之名，皆源自历代果位之主的行迹意象。修士铸造道基、显化神通，本质上是在仿照前人意象求道——走前人走过的路，逐步与果位共鸣
- **宗风**：重心性积累，逍遥超脱，然进境迟缓

### 2. 青冥登天道

- **要旨**：以命养性，经世致用
- **门径**：融入人道，借官位、国运以反哺性功
- **枢要**：
  - **玄印**：官印也，将修士命数接引国运
  - **官身即法身**：修为与官品相系，升官即破境
  - **持玄**：青冥修士称「持玄」，然上限受帝君境界所制
- **宗风**：进境迅疾，然与官位深相系缚

### 3. 黄庭内景道

- **要旨**：身即宇宙，万象由心
- **门径**：内观存想，开辟黄庭内景，唤醒身神
- **次第**：
  - 五脏各应一大身神，各显一系神通
  - 五脏神全，以内景直感果位，一步证玄君
- **宗风**：诡秘莫测，直指本源，凶险至极

### 4. 服气养性道

- **要旨**：性命双修，形神俱妙
- **门径**：性与命同步圆满，自练气起即与果位自然呼应
- **宗风**：所求甚苛，被尊为「万道总纲」。从未断绝，然天地变迁后门槛日高，能修成者于各时代皆寥寥无几

## 五、核心修行概念

### 灵窍

**灵窍者，人身小宇宙与外界大天地之间先天的感应交汇之枢机。**

- **有则唯一，无则绝无。** 灵窍是仙凡之别的根本门槛。有灵窍者方可感应天地灵机、引气入体，踏入修行；无灵窍者终其一生不可修行，无论才智多么出众
- **灵窍不决定道途。** 修士能修何种功法，取决于其所接触的传承与机缘，而非灵窍本身。紫府金丹道、青冥登天道、黄庭内景道——三途之选系于际遇，非系于灵窍
- **灵窍的品质**：灵窍有感应灵敏与迟钝之分。灵敏者引气顺畅、感应精微；迟钝者修行如逆水行舟。但品质只决定起点难易，不决定最终成就
- **伪灵窍**：无灵窍者本不可修行。然顶尖仙族嫡系可借助极其珍稀的「清炁灵物」进行**点化**，强行开启隐晦脆弱的伪灵窍。此法风险极高——点化失败则损伤魂魄，即使成功，伪灵窍亦远不如先天灵窍稳定，修行上限受限，且终生伴有灵机紊乱之隐患

### 道基与神通

- **道基**：筑基期于下丹田（气海）铸就，乃特定道途法则之显化
- **抬升**：破境紫府之时，道基沿十二重楼逆流而升，入上丹田（泥丸宫）之升阳府
- **显化**：于升阳府中以神识之火煅烧，道基熔化显为**神通**
- **对应关系**：道基与神通一一相应。如「君兑隅道基」→「君兑隅神通」

### 金性

- 修士毕生性命交融之所铸
- 坚固不坏，唯一无二
- 金丹境以五道圆满神通熔炼金性，借以感应果位

### 果位

- 天地业位，道之侧面的凝注，悬于虚空，待修士归位
- 果位有主则其道兴盛，法则显化；空悬则其道衰退，法则隐晦
- 证得果位者称**真君**（金丹境）
- 分**果位**（主位，唯一）、**闰位**（客位，跨道途而求）、**余位**（从位，本道途分润）
- 三者皆为天地所设，空悬待归，不因修士意志而生灭
- 果位意象可被后世修士「注释」增补 → 详见**八、果位详章**

### 参紫仙槛

- 专指紫府境自三神通突破至四神通（大真人）之绝大关隘
- 无数修士困顿此关，百年蹉跎，终至寿元耗尽

### 求金法则

- 五神通圆满，以金性感应、叩问相应果位
- 正位有主则强求必殒
- 求余位须正位真君默许
- 求闰位须「三同二殊」或「四同一殊」之神通格局，且须于两道途法则皆有深悟

## 八、果位详章

### 位阶三品

| 位阶           | 别称 | 说明                 |
| :------------- | :--- | :------------------- |
| **果位** | 主位 | 一道之统宗，唯一不二     |
| **闰位** | 客位 | 跨道途而求，位属他道 |
| **余位** | 从位 | 本道途分润，辅佐主位 |

### 闰位求取之则

- 修士以本道为基，跨求他道之位，所成者为他道之闰
- 闰位之名，以所属道途为纲，不以主修之道为称
  - 例：主修太阴，求寒炁闰位 → 功成后位属寒炁，称「寒炁闰」
  - 例：主修寒炁，求太阴闰位 → 功成后位属太阴，称「太阴闰」
- 闰位之求，须得所属道途果位之主允准；果位空悬则可自求感应

### 余位求取之则

- 余位为本道分润之位，求取须本道果位之主允准
- 果位空悬则余位法则不稳，求之甚难
- 余位之名以本道为称，如太阴余位、玄雷余位

### 位阶高下

- 大抵果位最尊，闰位次之，余位又次之
- 然修士个人意象之深浅、道行之厚薄、神通之精粗，皆可补位阶之不足
- 强闰可胜弱果，强余可胜弱闰

### 果位意象

#### 意象层积

果位意象非亘古不变，乃历代果主行迹所铸。

- **初始意象**：果位初生时所秉之根本权柄，天地所赋，亘古不易
- **增补意象**：后世修士证得此果，其平生功业、道行、乃至陨落之相，皆可化为新注，层积于果位之上

#### 意象之变

果位意象如同一本书，每任果主书写新章。新章分量越重，旧章越易褪色湮灭。

- **可累加**：后人可在前人意象之上增补新注，使果位权柄日益丰厚
- **可消磨**：果位久悬无主或其道衰微，旧有意象可渐次淡化，乃至湮灭
- **可攫夺**：后证者以惊世功业覆盖前人之注——若新篇章的分量压倒旧章，旧意象便褪色至消失，新意象独尊

攫夺在紫府层面的具体表现：被攫夺者的神通会随意象改变而变化。例如，若明阳果主攫夺了渌水"倾覆"之意象，则渌水一途的"天下覆"神通将转变为其他意象（如"如重浊"），而明阳一途则因此新增"天下明"神通。神通之名与意象绑定，意象易主则神通亦变。

#### 求道之则

- 修士求取果位，其道行、神通、功业，须与果位既有意象相契
- 相契愈深，感应愈易，归位愈顺
- 若强行求取意象相悖之果，轻则感应不彰，重则遭法则反噬，身死道消
