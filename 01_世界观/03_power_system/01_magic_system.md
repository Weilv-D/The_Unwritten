# 修行体系总纲

> 本文件为力量体系的哲学基础与核心机制，所有修行相关设定的基准。

## 一、性命二元论

### 总纲

修行乃是以个体之「性」去认知、交互，并最终承负宇宙定位之「命」的宏大工程。

### 性（内求之本）

- **构成**：法躯、修为、道行
- **来源**：一切通过**内炼**所获得的、属于**自我**的力量
- **关联**：直连**身寿**。性强则法躯坚固，长生久视
- **本质**：「我」的力量

### 命（外应之凭）

- **构成**：位格、命数、魂魄
- **来源**：一切通过**外感**所获得的、属于**世界定位**的权柄
- **关联**：直连**命寿**。命贵则位格崇高，劫难不侵
- **本质**：「世界」赋予「我」的力量

### 性命关系

- 性无命不立，命无性不灵
- 修行即性命双修，缺一不可
- 紫府境：性命初步交融，真魂凝聚
- 金丹境：性命融合为不朽金性，求得果位，天地同寿

## 二、三魂七魄

三魂七魄是「性」与「命」在人身中的具体化现。三魂属阳/性，七魄属阴/命。

## 四、四大修行体系

### 1. 紫府金丹道

- **核心理念**：以性求命，内圣外王
- **路径**：先炼性功至金丹圆满，再以此求取果位
- **关键步骤**：
  - 筑基期铸造**道基**（如「君兑隅道基」）
  - 紫府期将道基抬入升阳府，显化为**神通**
  - 五神通圆满后熔炼为**不朽金性**，感应果位，成就金丹
- **特质**：重心性积累，逍遥超脱，进境较慢

### 2. 青冥登天道

- **核心理念**：以命养性，经世致用
- **路径**：融入人道，借官位、国运反哺性功
- **关键概念**：
  - **玄印**：官印，将修士命数接入国运
  - **官身即法身**：修为与官品绑定，升官即破境
  - **持玄**：青冥修士称「持玄」，上限受帝君境界制约
- **特质**：进境迅捷，但与官位深度绑定

### 3. 黄庭内景道

- **核心理念**：身即宇宙，万象由心
- **路径**：内观存想，开辟黄庭内景，唤醒身神
- **关键步骤**：
  - 五脏对应五大身神，各显一系神通
  - 五脏神全后，以内景直接交感果位，一步证玄君
- **特质**：诡秘莫测，直指本源，凶险异常

### 4. 服气养性道

- **核心理念**：性命双修，形神俱妙
- **路径**：性与命同步圆满，从练气起即与果位自然呼应
- **特质**：要求苛刻，上古后几乎无人能成，被尊为「万道总纲」

## 五、核心修行概念

### 道基与神通

- **道基**：筑基期于下丹田（气海）铸造，是特定道途法则显化
- **抬升**：突破紫府时，道基沿十二重楼逆流抬入上丹田（泥丸宫）的升阳府
- **显化**：在升阳府中以神识之火煅烧，道基熔化显化为**神通**
- **对应关系**：道基与神通一一对应。如「君兑隅道基」→「君兑隅神通」

### 金性

- 修士毕生性命修为交融
- 坚固不坏，唯一无二
- 金丹境以五道圆满神通熔炼金性，以此感应果位

### 果位

- 天地业位，道之侧面的凝聚，悬于虚空，待修士归位
- 果位有主则其道兴盛，法则显化；空悬则其道衰退，法则隐晦
- 证得果位者称**真君**（金丹境）
- 分**果位**（主位，唯一）、**闰位**（客位，跨道途而求）、**余位**（从位，本道途分润）
- 三者皆为天地所设，空悬待归，不因修士意志而生灭
- 果位意象可被后世修士「注释」增补 → 详见**八、果位详章**

### 参紫仙槛

- 特指紫府境从三神通突破至四神通（大真人）的巨大瓶颈
- 无数修士困于此关百年，耗尽寿元

### 求金法则

- 五神通圆满后，以金性感应、叩问对应果位
- 正位有主则强求必死
- 求余位需正位真君默许
- 求闰位需「三同二殊」或「四同一殊」神通格局，且需对两道途法则有深悟

## 八、果位详章

### 位阶三品

| 位阶           | 别称 | 说明                 |
| :------------- | :--- | :------------------- |
| **果位** | 主位 | 一道之统宗，唯一     |
| **闰位** | 客位 | 跨道途而求，位属他道 |
| **余位** | 从位 | 本道途分润，辅佐主位 |

### 闰位求取之则

- 修士以本道为基，跨求他道之位，所成者为他道之闰
- 闰位之名，以所属道途为纲，不以主修之道为称
  - 例：主修太阴，求寒炁闰位 → 功成后位属寒炁，称「寒炁闰」
  - 例：主修寒炁，求太阴闰位 → 功成后位属太阴，称「太阴闰」
- 闰位之求，须得所属道途果位之主允准；果位空悬则可自求感应

### 余位求取之则

- 余位为本道分润之位，求取须本道果位之主允准
- 果位空悬则余位法则不稳，求之甚难
- 余位之名以本道为称，如太阴余位、玄雷余位

### 位阶高下

- 大抵果位最尊，闰位次之，余位又次之
- 然修士个人意象之深浅、道行之厚薄、神通之精粗，皆可补位阶之不足
- 强闰可胜弱果，强余可胜弱闰

### 果位意象

#### 意象层积

果位意象非亘古不变，乃历代果主行迹所铸。

- **初始意象**：果位初生时所秉之根本权柄，天地所赋，亘古不易
- **增补意象**：后世修士证得此果，其平生功业、道行、乃至陨落方式，皆可化为新注，层积于果位之上

#### 意象之变

- **可累加**：后人可在前人意象之上增补新注，使果位权柄日益丰厚
- **可消磨**：果位久悬无主或其道衰微，旧有意象可渐次淡化，乃至湮灭
- **可攫夺**：后证者以惊世功业覆盖前人之注，乃至取而代之，使旧意象湮灭，新意象独尊

#### 求道之则

- 修士求取果位，其道行、神通、功业，须与果位既有意象相契
- 相契愈深，感应愈易，归位愈顺
- 若强行求取意象相悖之果，轻则感应不彰，重则遭法则反噬，身死道消
