# 势力总览

> 势力流变谱系（辈分体系、一代弟子总表、四君体系、世家流变）详见 `02_势力流变谱系.md`。本文件为各势力之当世状态概览。

## 一、当世格局概览

三玄旧盟废后，天下复乱。仙宗相互攻伐，世家朝秦暮楚。此乱持续数百年，至当世犹未止。三玄主未曾决裂，弟子之争亦不曾明面撕破——三玄共檐之义，名义上从未断绝。

> 三玄你中有我、我中有你。当世多数仙宗混合了多玄传承，弟子往往不自知。以下按主要倾向归类，但不可理解为各势力"属于"某一玄。

当世势力可分六类：

| 类别 | 势力 | 主要传承倾向 | 势力层级 | 地域 |
|------|------|-------------|---------|------|
| 仙宗 | 各大宗门（存者不足十家） | 多玄混合 | 棋子级~棋手级 | 各地 |
| 神庭 | 北极驱邪院 | 兜玄为主 | 棋手级（隐） | 不明 |
| 混合宗门 | 太微玄宫 | 玉虚+清都+多玄 | 棋子级 | 东方 |
| 龙族 | 龙属 | 青玄渊源 | 棋手级 | 东海 |
| 外来修行 | 释修 | 渊源不明 | 棋子级 | 北方 |
| 世家 | 十余古世家 | 各有依附 | 消耗级~棋子级 | 各地 |

> 势力层级依据 `core-world.md` 之"势力分层博弈"定义：棋手级（金丹/真君）、棋子级（紫府/摩诃）、消耗级（筑基及以下）。

## 二、通玄系·仙宗

### 沿革

九真混成后，通玄弟子各立宗门，初时数百，经千年兼并定为三十六仙宗。天坠毁去近半，天下从革中再减，今存者不足十家。

三十六仙宗非定数，仅言其多。后世以为定数者，实为讹传。

### 九真后裔脉络

九真皆通玄·希字辈。详见 `02_势力流变谱系.md`。

| 九真 | 成就 | 脉络演变 | 当世状态 |
|------|------|---------|---------|
| 希阳真君 | 真君 | 开少阳脉 | 已陨落 |
| 希月真君 | 真君 | 开月宫脉 | 已隐世 |
| 希玄真君 | 仙君 → 天外 | 开玉虚脉 | 近古分裂：主脉西迁；东方余脉为太微玄宫 |
| 希辰真君 | 真君 | 开紫府脉 | — |
| 希清真君 | 仙君 → 天外 | 开清都脉（青玄主脉） | 三度废立终绝。余脉借壳为太微玄宫 |
| 希灵真君 | 真君 | 开洞华脉 | — |
| 三人佚名 | — | — | — |

### 仙宗内部结构（通则）

仙宗虽各有不同，然组织规制大同小异：
- **掌门**：真君或大真人，总领宗务
- **长老会**：紫府修士，参与决策
- **内门弟子**：筑基以上，核心传承
- **外门弟子**：练气为主，择优入内门
- **附属势力**：凡人国度、矿脉、灵田

### 与世家的关系

仙宗立宗传法，世家以血脉传道。二者相互依存又相互提防：
- 仙宗需要世家提供灵脉、矿藏、凡人基础
- 世家需要仙宗提供功法传承、护道之力
- 世家背后往往有仙宗支持，仙宗背后亦有世家资助
- 大晟之世即为世家间接统治的典型——晟朝皇室实为某世家分支

## 三、兜玄系·北极驱邪院

### 基本信息

| 项目 | 内容 |
|------|------|
| 别称 | 北院、驱邪院、神庭；释修称"帝释天"（讹传） |
| 性质 | 神庭——非宗门，类似天庭。北宫神雷，燮理天下，总摄鬼神 |
| 归属 | 兜玄祖庭 |
| 创立 | 上古初，仙神之争中 |
| 初代院主 | 执衡真君 |
| 核心建筑 | 祀灵坛（祀太古战死之神） |

### 历史

上古初立院，总摄天下鬼神。上古末宫移羽盖之变后封院三百年。中古派观复道人收集古道藏，事败。此后持续衰落，当世封院已久。

### 当世状态

- **公开认知**：已封院，院址不明，祀灵坛或已荒废
- **隐秘真相**：实未封。历代院主以秘法传承，在院中修行。祀灵坛中太古之神真灵尚存，不显于世
- **院中密室**：藏有正始观师像（自玉京山移来），历代院主口传

## 四、青玄系·太微玄宫

### 基本信息

| 项目 | 内容 |
|------|------|
| 别称 | 东玉虚、太微宗；玉虚主脉呼为"伪宫" |
| 归属 | 表面通玄（自称承玉虚正统），实则清微道统（清都余脉借壳） |
| 创立 | 近古中期 |
| 地域 | 东方 |

### 真实渊源

玉虚西狩后，东方余下的玉虚弟子联合部分世家立太微玄宫。表面自称承冲虚真君，实为清都观余脉借壳重生——功法表面玉虚一系，内核多有清都痕迹。

太微立后百年，内部"楼观宗圣之辨"：
- **楼观派**（主传法弘道，广收门徒）= 清都余脉 → 胜
- **宗圣派**（主隐修求道，不涉尘世）= 玉虚旧部 → 败

楼观派胜后清都余脉彻底掌控太微，势力大张，弟子出走别立宗门或入凡建世家，近古中后期仙宗格局多源于此。

### 内部分化

太微弟子自以为玉虚正宗，不知是清都余脉。此秘密仅极少数高层知晓。

## 五、青玄系·龙属

### 基本信息

| 项目 | 内容 |
|------|------|
| 始祖 | 龙蹻【别：蹻龙、龙祖；龙属呼"老祖公"】 |
| 归属 | 青玄（龙蹻为青玄主代师收徒，辈份极高） |
| 地域 | 东海（中古自南海北迁） |
| 势力层级 | 棋手级，当世最大势力之一 |

### 沿革

| 时期 | 事件 |
|------|------|
| 上古初 | 龙蹻入青玄门下，受化龙之法，开龙属一脉 |
| 上古 | 青玄主令龙属自立，不涉三玄内斗 |
| 中古初 | 天坠后南方受损尤重，青玄主授意北迁，入主东海 |
| 当世 | 据东海，为当世最大势力之一 |

### 特点

- 辈份极高：历代真君见龙蹻亦礼敬（代师收徒，相当于正始道人徒孙辈）
- 中立立场：名义上不涉三玄内斗，实则暗承青玄意旨
- 东海之主：原居东海之妖族或降或走

## 六、独立势力·释修

### 基本信息

天坠后自西方传来，入主北方，为当世新兴势力。不属三玄体系，修行路径与三玄正传迥异。

### 当世地位

- 在北方有实质影响力
- 与兜玄有教义冲突（称北极驱邪院为"帝释天"）
- 对三玄内争持观望态度
- 影响了近古以降的名号体系（"天尊"等称呼受释修影响）

---

## 七、世家

七十二世家为上古不愿立宗者，以血脉传道。此数目非实数，仅言其多。代有兴替，今世所谓"古世家"者不过十余姓。

世家特征：
- 以血脉传承功法，不公开收徒
- 各有依附的仙宗，亦被仙宗所依赖
- 部分世家入凡立国，大晟即其典型（然大晟实为兜玄主导，非纯世家）
- 大晟立昭元一府（居晟煌天），为有晟一朝的道统之法府，号"不为谁家道统"
- 晟朝亡后，昭元一府状态不明，天下从革中世家格局大洗牌

---

## 八、凡人国度

当世凡人国度林立，背后各有仙宗世家支持。凡人不知仙宗之名，不知三玄之别。只知山上有仙人、北方有和尚、东海有龙。

凡人国度于修士而言，是资粮来源（灵脉、矿藏、人口）与势力屏藩。仙宗不直接统治凡人，通过世家或代理间接控制——大晟之世即为典范（大晟以兜玄为根底，借皇权统一修行秩序，废三千二百宗）。
