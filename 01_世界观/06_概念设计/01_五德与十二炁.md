# 五德与十二炁

## 一、五德

### 概念

五德者，天地初判时由正始道人所定之五种根本属性。五德各有主者，皆太古神圣，名号失传。今世修士所谓"太古五主"，实为后人追称。

五德与天地业位体系中的**五行**（金木水火土）直接对应。五行二十五道（每行五子道），即五德在修行层面的展开。

### 五德之主

五德之主为太古神圣，仙神大战中战死其四，仅余水德之主的后裔——龙蹻。

| 德 | 对应五行 | 太古之主 | 当世遗存 |
|----|---------|---------|---------|
| 金德 | 庚金、兑金、库金、逍金、齐金 | 战死 | — |
| 木德 | 更木、正木、角木、保木、集木 | 战死 | — |
| 水德 | 渌水、坎水、牝水、府水、合水 | 战死 | 龙蹻（后裔） |
| 火德 | 灴火、离火、牡火、真火、并火 | 战死 | — |
| 土德 | 宣土、艮土、宝土、戊土、归土 | 战死 | — |

> "五德之中，水德最盛"——此为太古定论，亦是龙蹻入青玄门下的因缘。

### 五德与三德

五德为天地之根本属性，三德（仙德、神德、人德）为三玄对五德的不同理解路径：
- 通玄以五德为仙道之基 → 五行二十五道纳入紫府金丹道体系
- 兜玄以五德为神道之源 → 太古五主即神道之源头
- 青玄以五德为人道之资 → 正始之道的"性命同修"以五德为本

## 二、十二炁

### 概念

五德既定，十二炁从之而出。十二炁为天地间十二种根本炁机，上仪、下仪最先，真炁次之，余者渐次分判。

太古修士多兼修数炁，不似后世专精一道。初分次序今已失传。

### 与天地业位体系的关系

十二炁在天地业位体系中为独立的大道分类，每炁各含五神通：

| 炁 | 性质 |
|----|------|
| 清 | 纯净、澄明 |
| 邃 | 深远、幽冥 |
| 真 | 归真、本源 |
| 紫 | 紫气、祥瑞 |
| 晞 | 光明、破晓 |
| 寒 | 凛冽、封绝 |
| 谪 | 贬落、仙凡之隔 |
| 瑞 | 祥瑞、降泽 |
| 煞 | 杀伐、孤绝 |
| 华 | 繁盛、光华 |
| 上仪 | 天象、法度 |
| 下仪 | 地载、轨则 |

### 历史意义

十二炁的发现与五德的确定，标志着"正始"之世的完成。此后三玄之法皆以五德十二炁为根基。

南乡道藏中亦载有十二炁的相关记载，因其源出正始道人遗留，与三玄之法同源。

## 三、阴阳

阴阳为正始道人"剖混沌"之刀——非五德之一，而是五德分立之前提。

"天地初判，阴阳未分"→ 正始道人"以阴阳为刀"→ 阴阳始分，五德始显。

在天地业位体系中，阴阳为独立大道分类，含太阴、太阳、少阴、少阳、明阳、厥阴六道，与五行、八风、十二炁并列。
