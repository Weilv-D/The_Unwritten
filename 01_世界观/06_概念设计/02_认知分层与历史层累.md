# 认知分层与历史层累

## 一、认知分层

世间修士与凡人对世界的认知，因境界、门派、传承而有天壤之别。

### 四层认知

| 层级 | 群体 | 所知 | 所不知 |
|------|------|------|--------|
| 第一层 | 凡人 | 山上有仙人；北方有和尚；东海有龙；乱世求活 | 仙宗之名、三玄之别、五德十二炁 |
| 第二层 | 下修（练气筑基） | 三玄自古对立；三十六仙宗为定数；六天故气为魔功；自家宗门纯属某一玄 | 三玄同源；历史被篡改；功法多出古道藏；各宗多玄混合之实 |
| 第三层 | 上修（紫府以上） | 三玄曾共檐；天坠非自然；有观复道人其人 | 三玄为何而分；天坠谁所为；观复渊源；太微隐秘；历史远比三玄角力复杂 |
| 第四层 | 真君以上 | 部分真相：天坠为三玄主所为、观复背景复杂等 | 正始观所在、三玄主是否尚存、太古神圣真名、层累之下是否还有更深层 |

### 信息控制机制

| 机制 | 运作方式 |
|------|---------|
| 天地之禁 | 真君及以上，记载维艰。书其名则字迹易漫，绘其形则画幅易损，口传其事则辗转失真。非不可记，乃记载极易散佚——须以灵玉刻符、洞天封简等特殊手段方能久存，代价极高。故真君以上之名号、事迹，多只能口耳相传，且传多失真 |
| 历史篡改 | 各宗各派据立场增删改易史料，每代修史者皆上一层记载 |
| 真本封存 | 古代典籍真本藏于洞天，下修无缘得见 |
| 口耳相传 | 最核心的秘密不立文字，唯真君口传（亦因天地之禁，真君名号只能口传） |
| 制造讹传 | 刻意制造"灵宝之争""弟子私怨"等说法，掩盖复杂因果 |
| 阵营简化 | 将历史中多方势力的复杂角逐简化为"三玄对立"或"仙神正邪"，掩盖你中有我的真相 |
| 神话化 | 凡人层面的记忆被压缩为神话传说（盘古、女娲、共工等） |
| 溯源不可靠 | 即便【隐】层级的"真相"也可能是更早的修史者编造，层累无底 |

## 二、历史层累

同一事件，不同时代有不同记载。历代修史者，各据立场增删改易。故今日所传之"史"，实为层累而成。

### 层累示例：仙神之争

| 时期 | 叙事版本 | 修史者 |
|------|---------|--------|
| 太古末（当时） | 三玄弟子因多种原因爆发冲突（利益、地盘、道统解释权等） | — |
| 上古（第一次修史） | 通玄兜玄弟子私怨 | 青玄弟子删去本门参与记录 |
| 中古（第二次修史） | 争夺灵宝 | 兜玄主加入"灵宝"细节以掩真相 |
| 近古（第三次修史） | 仙神正邪之争 | 通玄修史者加入"正邪"之说，崇仙抑神 |
| 当世流传 | 第三次修史之版本 | 洞天中尚存前两次之残篇 |

### 层累的后果

- 下修所学"历史"实为各宗有意编造
- 同一事件在不同宗门有截然不同的记载
- 古代真相的线索散落各处，无人能拼全貌
- 即便真君，也只能窥见部分真相

### 对叙事的启示

> 此认知分层机制意味着：故事中的任何角色，其所知所信都可能是错的。角色基于错误认知做出的决策，才是真正的戏剧冲突来源。读者随着视角人物逐步发现"原来历史不是这样"，构成世界观的渐进式揭示。
