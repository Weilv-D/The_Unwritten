# 核心意象与符号体系

## 五卷诗经意象延伸

### 卷一·蒹葭苍苍——水·雾·霜

| 维度 | 设计 |
|------|------|
| **主色调** | 灰白（霜/雾/雪）+ 暗红（血/铁锈/矿火） |
| **核心意象** | 蒹葭（芦苇）——根基浅薄，随风摇曳，看似脆弱却能在淤泥中生长。霜——寒冷、遮蔽、令一切模糊不清。水——无处不在却抓不住 |
| **视觉符号** | 枯芦苇丛、结冰的河面、矿洞深处的地下水、雾中模糊的人影 |
| **听觉符号** | 北风的呼啸、冰面碎裂的声响、矿洞深处的闷响 |
| **触觉符号** | 刺骨的寒、灵石粗糙的表面、铁器冰冷的握感 |
| **情感基调** | 模糊、不确定、在混沌中摸索前行。"所谓伊人，在水一方"——他寻找的东西始终看不清 |

### 卷二·交交黄鸟——棘·笼·金

| 维度 | 设计 |
|------|------|
| **主色调** | 金赤（宗门/权力/庚金）+ 青蓝（河洛/水系/深潭） |
| **核心意象** | 黄鸟止于棘——一只鸟停在荆棘上，无处可栖，进退维谷。棘——规则的刺、势力的边界、选择的代价。笼——宗门是笼，也是庇护 |
| **视觉符号** | 棘刺围栏、宗门的层层门禁、金色与青色交织的权力纹饰、案卷中被涂去的名字 |
| **听觉符号** | 棋子落盘声、典籍翻页声、密室中的低语 |
| **触觉符号** | 厚重典籍的质感、灵脉石温热的表面、丝绸与麻布的对比 |
| **情感基调** | 受困、博弈、明暗交织。"止于棘"——有了栖身之处，但处处是刺 |

### 卷三·悠悠苍天——废墟·灰烬·镜

| 维度 | 设计 |
|------|------|
| **主色调** | 玄紫（秘密/遗迹/太阴）+ 苍绿（古楚/巫术/云梦泽） |
| **核心意象** | 黍离之悲——站在废墟上追问"彼何人斯"。废墟——玄雷宫遗迹、古楚遗迹、被涂抹的历史。镜——照见自己体内的异力、照见被掩埋的真相 |
| **视觉符号** | 倒塌的石柱、墙壁上被凿去的铭文、铜镜中诡异的倒影、水底的残垣 |
| **听觉符号** | 遗迹中的回声、古籍翻动时纸页碎裂的声响、雷鸣（异力涌动时） |
| **触觉符号** | 古老石壁的冰冷、灰烬的干涩、铜镜的冰凉表面 |
| **情感基调** | 追问、迷失、真相浮现时的震动。"此何人哉"——我是谁？这世界是什么？ |

### 卷四·大风有隧——火·铁·尘

| 维度 | 设计 |
|------|------|
| **主色调** | 焦黑（战火/毁灭）+ 暗金（权力/代价）+ 各色（多视角多阵营） |
| **核心意象** | 大风有路——一场席卷天下的风暴。贪人败类——每个"为了更大的善"而行动的人，最终都成了败类。隧——风的通道，也是死路 |
| **视觉符号** | 燃烧的洞天、崩塌的法阵、战场上密密麻麻的尸体、天纲裂痕中的异色光芒 |
| **听觉符号** | 战鼓、法术碰撞的轰鸣、洞天崩塌时天地震颤的巨响、沉默（战斗间隙的死寂） |
| **触觉符号** | 灼烧、震动、血与泥的混合 |
| **情感基调** | 毁灭、代价、没有正确答案。"贪人败类"——每一方都贪，每一方都在败 |

### 卷五·如月之恒——月·新生·门

| 维度 | 设计 |
|------|------|
| **主色调** | 月白（太阴/新生）+ 晨曦色（微弱的暖光） |
| **核心意象** | 月渐满——残月渐圆，旧秩序终结时新事物在生长。日方升——但不是旧日的太阳，是新的光。门——通向天外的门，通向新天纲的门，通向选择的大门 |
| **视觉符号** | 月光、新芽、打开的门、站在天地之间的人影 |
| **听觉符号** | 寂静——全书的收束不是轰鸣，是"暮云四垂，八风不应"的静 |
| **触觉符号** | 清冷但不刺骨的风、新土地的柔软 |
| **情感基调** | 沉静、抉择、不是胜利而是回答。"如月之恒，如日之升"——旧的月亮落下，新的月亮升起 |

## 全书核心象征物

### 玉（京）
玉京——仙人居所，也是全书标题的核心意象。玉在本书中象征"秩序"——看似温润，实则冰冷坚硬、不可动摇。坠玉京——秩序不是被外力打碎的，是从内部开始坠落。

### 雷（玄雷）
雷象征"天罚"——公正的暴力。但雷本身无善恶，只是法则的执行。玄雷宫的覆灭意味着"天罚"的缺席——此后两千年，天纲的执法者消失，三玄争端再无制约。

### 阴阳（太阴/太阳）
太阳陨而太阴沉默——旧秩序的两根支柱先后倒塌。阴阳不是善恶，是循环。打破循环意味着打破阴阳交替的自然法则。

### 三棵树
服气养性道是一棵大树，天变后分裂为三棵——各自带着原树的基因但再也无法合为一体。这是"三德之争"的物理隐喻。

### 镜
铜镜在本书中反复出现——陆无咎的铜尺、明重华的法器铜镜、古楚遗迹中的铜镜碎片。镜象征"照见真相"——但每面镜子照出的都是不同层面的真相。

## 命名美学

### 地名命名
- **实意命名**：蓟城、洛邑、襄阳等真实地名沿用
- **意象+地理**：昆兰山、太华、天柱山等
- **仙道命名**：紫霄天、太乙神刑天等洞天

### 功法命名
- **道统+意象**：【紫府庚金诀】【黄庭存神法】
- **古风简练**：四个字为主，用【】包裹

### 人物命名
- **主角组**：含寓意的古典姓名
- **配角**：普通但有质感的名字
- **凡人**：朴实、贴近生活的名字
- **古修**：道号/法号风格

### 章节标题
- 四字或六字古风意象
- 暗示本章核心事件但不过度剧透
- 同卷内不重复用字
