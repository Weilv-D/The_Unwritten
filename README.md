# The Unwritten

> 一部百万字级别的超长篇小说创作项目

---

## 项目简介

**The Unwritten** 是一个专业的小说创作工作空间，专为**超长篇小说（百万字级别）**的系统性创作而设计。

本项目采用最佳实践，整合了：
- **文学创作理论**：三幕剧结构、英雄之旅、救猫咪等经典理论
- **AI协作写作**：优化Claude Code的交互体验
- **版本控制管理**：使用Git进行专业的版本控制
- **超长篇优化**：专门针对百万字规模的特殊挑战

### 核心理念

1. **结构化组织**：清晰的层级关系，从世界观→人物→大纲→正文
2. **模块化创作**：每个文件独立聚焦，避免过度耦合
3. **一致性维护**：系统化的人物、世界观、剧情一致性检查
4. **可追溯性**：完整的修改历史和决策记录
5. **AI友好**：优化与Claude Code的协作体验

## 常见问题

### Q: 为什么要这么复杂的文件结构？
**A**: 百万字小说的内容量巨大，需要系统化组织才能保持一致性和可维护性。清晰的分类和索引能快速定位内容，避免在几百个文件中迷失。

### Q: 文件大小限制是什么？
**A**:
- 章节正文：≤5000字
- 场景文件：≤3000字
- 人物设定：≤3000字
- 世界观设定：≤5000字

超过限制则拆分为多个文件。

### Q: 如何确保人物一致性？
**A**:
1. 使用人物出场追踪表记录每次出场
2. 定期使用一致性检查提示词验证
3. 修改人物前必须检查现有设定
4. 新特征需更新到人物档案

### Q: 剧情线索如何追踪？
**A**:
1. 使用剧情线索追踪文件记录主线和支线
2. 标注关键节点和完成状态
3. 记录线索之间的交织关系
4. 定期更新线索进展

### Q: 如何进行周期性审查？
**A**:
1. **每周**：使用一致性检查提示词检查当周章节
2. **每月**：完整审查当卷内容，生成一致性报告
3. **每卷**：全面审查，创建版本标签，写卷次摘要
