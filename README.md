# The Unwritten — 未书

> 百万字级别仙侠权谋超长篇小说创作工作空间

---

## 项目简介

**未书**（The Unwritten）是一部仙侠 × 权谋 × 群像小说，5卷，每卷约45章×6000字，时间跨越两百年。

风格参照：百年孤独（家族兴衰+宿命轮回）、白鹿原（魔幻现实）、冰与火之歌（POV+残酷博弈）。

核心主题：长生需争，旧秩序的崩塌，没有背景的人没有任何希望。

## 风格规范速查

- **文风**：半文半白，古典简练，严禁现代口语
- **段落公式**：[景-人-心] 三段式递进
- **内心独白**：『』包裹，表里反差（嘴上恭敬，心里算计）
- **12条禁止规则**：详见 `AGENTS.md` 第三节
- **节拍器理论**：十大序列 × 八大要素，详见 `AGENTS.md` 第四节
