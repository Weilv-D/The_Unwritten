# 人物类提示词模板

> 用于创建、发展和检查人物的标准化提示词

---

## 1. 人物生成

### 使用场景
需要创建新的故事人物时

### 输入格式
```xml
<context>
  <character_type>主角/配角/反派/群演</character_type>
  <story_role>[在故事中的功能]</story_role>
  <world_setting>[世界观设定路径]</world_setting>
  <existing_characters>[已有人物列表，避免重复]</existing_characters>
  <special_requirements>[特殊要求]</special_requirements>
</context>
```

### 任务要求
请生成完整的人物档案，包含：

1. **基本信息**（姓名、年龄、外貌特征、身份）
2. **性格分析**（优点、缺点、矛盾点、怪癖）
3. **内在需求**（表层目标+深层渴望+阻碍）
4. **背景故事**（关键成长事件、创伤、获得技能的方式）
5. **对话风格示例**（2-3句典型对话）
6. **人物弧光**（起始状态→成长→最终状态）

### 生成原则
- 人物必须有矛盾性（优点和缺点并存）
- 必须有明确的动机（表层和深层）
- 必须与世界观设定一致
- 必须与其他人物有差异化
- 避免刻板印象和陈词滥调

## 2. 人物对话风格设计

### 使用场景
需要为人物设计独特的对话风格

### 输入格式
```xml
<context>
  <character>[人物名称]</character>
  <personality>[性格关键词]</personality>
  <background>[背景简述]</background>
  <education>[教育水平]</education>
  <region>[出身地区]</region>
</context>
```

### 任务要求
请设计人物的对话风格，包含：

1. **语言特点**（正式/随意、简洁/啰嗦、文雅/粗犷）
2. **口头禅**（1-2个标志性用语）
3. **语气特点**（命令/协商/询问/讽刺...）
4. **表达习惯**（比喻方式、逻辑方式...）
5. **情绪表现**（如何表达愤怒/悲伤/喜悦...）
6. **对话示例**（3种不同场景下的对话）

### 差异化原则
- 每个人物的说话方式必须可区分
- 读者仅通过对话就能识别是谁在说话
- 对话风格必须与性格和背景一致

## 3. 人物关系设计

### 使用场景
需要设计两个或多个人物之间的关系

### 输入格式
```xml
<context>
  <character_a>[人物A的设定路径]</character_a>
  <character_b>[人物B的设定路径]</character_b>
  <relationship_type>[预期关系类型]</relationship_type>
  <story_function>[在故事中的功能]</story_function>
</context>
```

### 任务要求
请设计人物关系，包含：

1. **关系本质**（表面关系vs真实关系）
2. **关系起源**（如何相识/产生关联）
3. **关系发展**（将如何变化）
4. **冲突来源**（关系中潜在的摩擦点）
5. **相互影响**（如何互相改变对方）
6. **关键场景**（展现关系的关键时刻）

## 4. 人物弧光设计

### 使用场景
需要设计人物的成长/变化轨迹

### 输入格式
```xml
<context>
  <character>[人物名称和当前设定]</character>
  <arc_type>正面弧光/负面弧光/平坦弧光</arc_type>
  <story_length>[故事总长度/章节数]</story_length>
  <theme>[与人物弧光相关的主题]</theme>
</context>
```

### 任务要求
请设计人物弧光，包含：

1. **起始状态**（初始信念、核心缺陷、表面形象）
2. **错误信念**（人物认为正确但实际错误的想法）
3. **成长触发**（什么事件开始改变人物）
4. **挣扎过程**（人物如何抵抗改变）
5. **转折点**（最终接受改变的关键时刻）
6. **最终状态**（新信念、新行为、新形象）
7. **关键章节**（弧光的关键节点分布在哪些章节）

### 弧光类型说明
- **正面弧光**：人物克服缺陷，获得成长
- **负面弧光**：人物被缺陷吞噬，走向堕落
- **平坦弧光**：人物坚持信念，改变周围世界
