# 周期性审查流程

> 每周/每月进行的一致性审查流程

---

## 每周审查（周日进行）

### 适用范围
审查本周完成的所有章节（通常3-5章）

### 审查步骤

#### 1. 人物一致性审查
对本周每章进行人物检查：

```xml
使用 07_提示词/一致性检查.md 中的「人物一致性检查」模式
```

检查要点：
- 语言风格是否符合设定（半文半白、敬称体系）
- 行为是否符合性格和动机
- 内心独白是否与外在言行形成表里反差
- 能力使用是否在设定范围内
- 称谓使用是否正确（真君/真人/大人/前辈...）

#### 2. 文风一致性审查
检查是否符合写作要求.md中的风格标准：

- [ ] 无直接情绪告知
- [ ] 无作者评论
- [ ] 视角固定（无上帝视角）
- [ ] 对话有信息推进
- [ ] 内心独白使用『』
- [ ] [景-人-心]段落结构
- [ ] 术语系统一致（『』和【】使用正确）
- [ ] 半文半白风格统一

#### 3. 时间线验证
- 检查本周章节间的时间流逝是否合理
- 季节/天气是否匹配
- 人物位置是否合理

#### 4. 伏笔记录更新
- 本周设置了哪些新伏笔
- 本周回收了哪些伏笔
- 更新伏笔追踪表

#### 5. 生成审查报告
将审查结果记录到：
```
06_追踪/审查报告/YYYY-MM-weekly.md
```

---

## 每月审查（月末进行）

### 适用范围
审查当月完成的所有内容（通常15-20章）

### 审查步骤

#### 1. 完整一致性检查
使用全面一致性检查提示词：
```xml
使用 07_提示词/一致性检查.md 中的「全面一致性检查（卷次级别）」模式
```

#### 2. 剧情节奏分析
使用节奏分析提示词：
```xml
使用 07_提示词/审阅提示词.md 中的「节奏分析」模式
```

检查要点：
- 节拍器理论的十大序列是否正确对应
- 情感曲线是否起伏合理（正负主题交替）
- 是否有平缓/脱钩的章节
- 每章是否完成至少一个核心叙事功能

#### 3. 人物弧线验证
- 主要人物本月是否有成长/变化
- 人物弧光是否按计划推进
- 角色决策是否基于利益动机

#### 4. 世界观扩展检查
- 新增设定是否与既有设定矛盾
- 涟漪展开是否按层级递进
- 冰山策略是否执行（只显露10%）

#### 5. 更新追踪表
- 全面更新人物出场追踪表
- 更新剧情线索追踪
- 更新伏笔追踪表
