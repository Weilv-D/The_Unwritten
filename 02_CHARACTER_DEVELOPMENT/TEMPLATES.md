# 人物设定模板

## 使用说明
创建新人物时，复制此模板到对应的人物文件夹中。

**重要提醒**（超长篇专用）：
- 每个人物文件夹顶部必须创建`SUMMARY.md`（200字内关键信息摘要）
- 所有人物设定文件控制在3000字以内
- 新人物创建后必须更新`00_appearance_tracker.md`

## 创建人物清单

创建新人物时，请按以下步骤操作：

1. **确定人物类型和文件夹**：
   - 主角：`01_protagonists/character_XXX_protagonist_[姓名]/`
   - 配角：`02_supporting/character_XXX_supporting_[姓名]/`
   - 反派：`03_antagonists/character_XXX_antagonist_[姓名]/`

2. **复制本模板**到对应文件夹

3. **创建SUMMARY.md**（超长篇关键）：
   ```markdown
   # [人物姓名] - 关键信息摘要

   ## 一句话概括
   [用一句话描述这个人物]

   ## 核心特征
   - 年龄：[年龄]
   - 身份：[身份]
   - 性格关键词：[关键词1]、[关键词2]、[关键词3]
   - 核心冲突：[内在冲突]

   ## 重要关系
   - 与[人物A]：[关系]
   - 与[人物B]：[关系]

   ## 角色弧光
   起始：[起始状态] → 结束：[结束状态]

   ## 语言风格
   [一句话描述说话方式]

   ## 首次出场
   第XXX章：[场景描述]
   ```

4. **填写模板内容**

5. **更新追踪表**：
   - 更新`02_CHARACTER_DEVELOPMENT/INDEX.md`
   - 更新`02_CHARACTER_DEVELOPMENT/00_appearance_tracker.md`
   - 如果是核心人物，更新`00_relationship_map.md`

6. **提交到Git**：
   ```bash
   git add 02_CHARACTER_DEVELOPMENT/
   git commit -m "feat(character): 创建新人物[姓名]"
   ```
