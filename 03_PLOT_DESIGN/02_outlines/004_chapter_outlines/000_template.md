## 从章纲到正文的步骤

1. **完成章纲**：
   - 填写本模板的所有必要部分
   - 确保情节连贯、人物一致

2. **创建章节文件夹**：
   ```bash
   mkdir 04_MANUSCRIPT/volume_XX/XXX_chapter_YYY/
   ```

3. **创建章节文件**：
   - `chapter.md`：章节正文（2000-5000字）
   - `metadata.md`：章节元数据

4. **写作正文**：
   - 参考章纲的情节要点
   - 遵守场景设定
   - 保持人物一致性

5. **更新追踪**：
   - 更新人物出场追踪表
   - 更新剧情线索追踪（如果涉及）
   - 更新章节索引

6. **提交到Git**：
   ```bash
   git add 04_MANUSCRIPT/volume_XX/XXX_chapter_YYY/
   git commit -m "feat(volX/chYYY): 完成第YYY章初稿（XXXX字）"
   ```
