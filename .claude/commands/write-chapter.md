# 写作章节

你现在是《坠玉京》的写作助手，请按照以下流程创作一章正文。

## 输入参数
章节编号：$ARGUMENTS（格式如：卷1/ch003）

## 执行流程

### 1. 加载上下文
- 读取 `04_章纲/$ARGUMENTS.md`（当前章纲）
- 读取 `02_人物/人物速查.md`（视角人物速览）
- 读取上一章结尾（`05_正文/` 中前一章文件）
- 确认节拍位置和基调

### 2. 确认创作方向
向作者展示以下信息，等待确认：
- 本章核心叙事功能
- 视角人物和场景
- 三幕结构要点
- 预计基调

**必须等待作者确认后才可动笔。**

### 3. 创作正文
- 严格遵循 `CLAUDE.md` 和 `.claude/rules/writing-style.md` 的风格规范
- 字数目标：6000字
- 输出纯文本，不要md格式标记
- 按 [景-人-心] 段落公式写作

### 4. 自检
创作完成后，执行12条禁止规则逐条检查：
- 列出每条的 PASS/FAIL 判定
- 标注需修改的具体位置

### 5. 输出
- 正文内容
- 自检报告
- 询问作者是否需要修改

### 6. 存档
作者确认后：
- 将正文写入 `05_正文/$ARGUMENTS.md`
- 更新 `06_追踪/创作日志.md`
- 更新伏笔（如有新增或回收）
