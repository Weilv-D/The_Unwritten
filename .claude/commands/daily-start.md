# 每日创作启动

按照以下流程为今日创作做准备。

## 执行步骤

### 1. 读取项目状态
- 读取 `00_项目基础/进度追踪.md`
- 读取 `00_项目基础/任务清单.md`

### 2. 汇报当前状态
向作者展示：
- 整体进度百分比
- 当前所处阶段（世界观/人物/大纲/章纲/正文）
- 待处理任务列表
- 上次会话的创作日志摘要

### 3. 确认今日目标
询问作者：
- 今天要做什么？（写作/审阅/设定/规划）
- 目标章节或任务
- 预计时长

### 4. 预加载上下文
根据今日目标自动加载：
- 写作：章纲 + 人物速查 + 上一章结尾
- 审阅：待审正文 + 对应章纲
- 设定：相关现有设定文件
- 规划：大纲 + 已有设定

### 5. 就绪确认
告知作者上下文已加载完毕，列出已读取的关键文件，准备开始工作。

---

**注意**：如果进度追踪.md中的数据与实际情况不符，先向作者确认再继续。
