# 审阅章节

对已完成章节进行全面质量评估。

## 输入参数
章节编号：$ARGUMENTS（格式如：卷1/ch003）

## 执行流程

### 1. 加载材料
- 读取 `05_正文/$ARGUMENTS.md`
- 读取 `04_章纲/$ARGUMENTS.md`
- 读取视角人物档案

### 2. 七维评估（每项1-10分）

1. **情节推进**：章纲规划点是否全部覆盖，推进是否自然
2. **人物表现**：行为是否符合设定，对话风格是否一致
3. **场景描写**：环境具象化，五感调动，氛围到位
4. **对话质量**：推动情节，符合性格，潜台词丰富
5. **节奏控制**：张弛有度，无拖沓或仓促
6. **文笔质量**：用词精准，无陈词滥调，句式有变化
7. **逻辑自洽**：时间线正确，因果关系合理

### 3. 12条禁止规则复查
逐条检查 PASS/FAIL，标注具体违规位置。

### 4. 输出报告

```
## 审阅报告：$ARGUMENTS

### 总分：X/10

### 各维度
- 情节推进：X | 评语
- 人物表现：X | 评语
- ...

### 优点
- ...

### 需修改（按优先级）
1. [位置] 问题 | 修改建议
2. ...

### 禁止规则违规
- 规则X：[位置] [说明]
```

### 5. 询问
询问作者是否需要根据报告修改。
