# 一致性检查

对指定章节或卷次进行系统化一致性审查。

## 输入参数
范围：$ARGUMENTS（单章如 `卷1/ch003`，全卷如 `卷1`）

## 执行流程

### 1. 确定检查范围
- 单章：仅检查该章
- 全卷：检查该卷所有已有章节

### 2. 五维一致性检查

#### A. 人物一致性
- 语言风格是否与设定一致
- 行为选择是否符合性格
- 能力使用是否在设定范围内
- 称呼是否正确
- 外貌描写是否一致

#### B. 世界观一致性
- 地理/距离是否正确
- 势力描述是否矛盾
- 力量体系使用是否合规
- 文化/礼仪是否恰当

#### C. 时间线一致性
- 章节间时间流逝是否合理
- 季节天气是否匹配
- 人物位置是否冲突
- 信息传递时间是否合理

#### D. 伏笔一致性
- 新设伏笔是否记录
- 已回收伏笔是否标注
- 长期伏笔状态是否更新

#### E. 物品/细节一致性
- 物品是否凭空出现/消失
- 人物知识是否超范围
- 伤势恢复是否合理

### 3. 输出报告

```
## 一致性检查报告：$ARGUMENTS

### 总分：X/10
### 发现问题：X（严重X / 中等X / 轻微X）

#### ✅ 通过项
- ...

#### ⚠️ 可疑项
- [位置] 描述 | 设定原文 | 建议

#### ❌ 错误项
- [位置] 描述 | 设定原文 | 修正建议

#### 优先修复
1. ...
```

### 4. 更新追踪文件
如有新发现，更新 `06_追踪/伏笔追踪.md`。
