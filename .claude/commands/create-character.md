# 创建人物档案

协助创建一个完整的人物档案。

## 输入参数
角色名称标识：$ARGUMENTS（格式如：主角_江既白、反派_某某、配角_某某）

## 执行流程

### 1. 收集基本信息
向作者询问（逐项确认）：
- 角色定位（主角/核心配角/反派/功能性配角）
- 核心一句话定位（"XX是什么样的人"）
- 初始登场卷次和场景
- 与其他角色的关系
- 特殊要求

### 2. 提供方案
基于收集信息，提供 2-3 个角色设计方案供作者选择，每个方案包含：
- 表层身份 + 隐藏身份
- 核心欲望 + 致命弱点
- 3-5个关键关系节点
- 人物弧光方向

**等待作者选定方案。**

### 3. 填充档案
按 `02_人物/角色模板.md` 格式创建完整档案：
- 基本信息（姓名、年龄、外貌、身份）
- 性格分析（优点、缺点、矛盾点、怪癖）
- 内在需求（表层目标+深层渴望+阻碍）
- 背景故事（关键成长事件）
- 对话风格示例（3种场景）
- 人物弧光（起始→成长→最终）
- 核心关系图谱

### 4. 一致性检查
- 与已有角色是否重复/矛盾
- 与世界观是否冲突
- 对话风格是否可区分

### 5. 写入文件
将档案写入 `02_人物/$ARGUMENTS.md`
更新 `02_人物/人物速查.md`
更新 `02_人物/INDEX.md`

### 6. 确认
展示完整档案，请作者审阅确认。
