# 灵感速记

快速记录创作灵感，自动分类归档到对应目录。

## 输入参数
灵感内容：$ARGUMENTS（自由文本，一句话或一段描述）

## 执行流程

### 1. 记录灵感
立即将灵感原话写入，不做丢失。

### 2. 分类识别
自动判断灵感属于哪个类别：

| 类别 | 归档路径 |
|------|----------|
| 人物（性格/对话/关系/弧光） | `02_人物/` 相关档案附录 |
| 世界观（设定/规则/地理） | `01_世界观/` 相关文件附录 |
| 剧情情节（事件/转折/悬念） | `08_灵感/剧情灵感.md` |
| 场景画面（环境/氛围/意象） | `08_灵感/场景灵感.md` |
| 对话片段 | `08_灵感/对话碎片.md` |
| 主题思考 | `08_灵感/主题思考.md` |
| 结构方法（节奏/章法） | `08_灵感/技法笔记.md` |
| 不确定/混合 | `08_灵感/待分类.md` |

### 3. 扩展（可选）
根据灵感内容，简要提示：
- 这条灵感与哪个已有设定/人物/伏笔相关
- 可能插入的卷次位置
- 需要额外设定什么才能落地

**但不强行扩展**——如果只是闪念，记录即可。

### 4. 写入
- 灵感原文 + 分类 + 时间戳 + 可选扩展
- 写入对应文件（追加到末尾）
- 更新 `08_灵感/INDEX.md`

### 5. 确认
告知作者灵感已记录到哪个文件。
