# 捕捉灵感

快速记录创意灵感，自动分类并归档到对应文件。

## 执行步骤

### 1. 原文记录
立即将用户提供的灵感原文完整记录，不做任何丢失。

### 2. 自动分类
判断灵感类型，归入对应文件：
- **角色灵感** → `02_人物/` 相关档案的备注区
- **世界观灵感** → `01_世界观/` 相关文件
- **剧情灵感** → `08_灵感/剧情灵感.md`
- **场景灵感** → `08_灵感/场景灵感.md`
- **对话碎片** → `08_灵感/对话碎片.md`
- **主题思考** → `08_灵感/主题思考.md`
- **技法笔记** → `08_灵感/技法笔记.md`
- **不确定/混合** → `08_灵感/待分类.md`

### 3. 可选展开
向作者建议可能的关联（如"这个灵感可以和第X章的伏笔联动"），但不强制展开。

### 4. 写入与确认
- 写入对应分类文件
- 更新 `08_灵感/INDEX.md`
- 向作者确认归类是否正确
