# 审阅章节

对已完成章节进行全面质量评估。

## 输入参数
章节编号：$ARGUMENTS（格式：卷1/ch003）

## 执行流程

### 1. 加载材料
- `05_正文/$ARGUMENTS.md`
- `04_章纲/$ARGUMENTS.md`
- 视角人物档案（`02_人物/人物速查.md`）
- **必读规则**：`writing-style.md`、`forbidden-rules.md`、`methodology-writing.md`、`methodology-character.md`

### 2. 七维评估（每项1-10分）

| 维度 | 要点 |
|------|------|
| 情节推进 | 章纲点是否覆盖，推进是否自然 |
| 人物表现 | 行为/对话是否符合设定 |
| 场景描写 | 五感调动，氛围到位 |
| 对话质量 | 推动情节，潜台词丰富 |
| 节奏控制 | 张弛有度，无拖沓 |
| 文笔质量 | 用词精准，句式变化 |
| 逻辑自洽 | 时间线/因果/人物位置 |

### 3. 12条禁止规则复查
逐条 PASS/FAIL，标注具体违规位置和原文。

### 4. 输出报告

```
## 审阅报告：$ARGUMENTS
### 总分：X/10
### 各维度评分和一句话评语
### 优点（2-3条）
### 需修改（按优先级排序）
### 禁止规则违规清单
```

### 5. 询问
询问作者是否需要根据报告修改。如需修改，直接定位到具体段落进行调整。
