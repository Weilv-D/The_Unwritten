# 设计卷次大纲

使用节拍器理论设计一卷的完整大纲。

## 输入参数

卷次：$ARGUMENTS（如 卷1）

## 执行流程

### 1. 加载基础材料

- `03_大纲/故事梗概.md`
- `01_世界观/INDEX.md` 及已有世界观文件
- `02_人物/人物速查.md`
- 已有卷大纲（如果是后续卷）
- `.claude/rules/creative-philosophy.md`

### 2. 确认本卷定位

**必须与作者讨论确认**：

- 本卷在全书中的战略功能（主题验证/势力重组/状态跃迁）
- 核心冲突是什么
- 主角本卷的起点和终点状态
- 本卷涉及的核心主题
- 预计章节数和字数

### 3. 设计十大序列框架

为每个序列填写八大要素：

| 序列     | 身份 | 欲望 | 动作 | 问题(内) | 阻力(外) | 结果 | 正负 |
| -------- | ---- | ---- | ---- | -------- | -------- | ---- | ---- |
| 开场     | ...  | ...  | ...  | ...      | ...      | ...  | ...  |
| 切入     | ...  | ...  | ...  | ...      | ...      | ...  | ...  |
| 激励     | ...  | ...  | ...  | ...      | ...      | ...  | ...  |
| 进展     | ...  | ...  | ...  | ...      | ...      | ...  | ...  |
| 转折     | ...  | ...  | ...  | ...      | ...      | ...  | ...  |
| 二次转折 | ...  | ...  | ...  | ...      | ...      | ...  | ...  |
| 危机     | ...  | ...  | ...  | ...      | ...      | ...  | ...  |
| 导入     | ...  | ...  | ...  | ...      | ...      | ...  | ...  |
| 高潮     | ...  | ...  | ...  | ...      | ...      | ...  | ...  |
| 结局     | ...  | ...  | ...  | ...      | ...      | ...  | ...  |

**提供 2-3 个方案供作者选择。**

### 4. 设计支线网络

为每条支线定义：

- 目的（主题探讨/势力展示/情报提供）
- 绑定视角人物
- 切入点、关键信息、与主干的勾连
- 收束方式和钩子

### 5. 规划伏笔

- 本卷设置的新伏笔
- 回收的前卷伏笔
- 跨卷长线伏笔的推进

### 6. 输出

- 完整大纲写入 `03_大纲/卷次/$ARGUMENTS_大纲.md`
- 更新 `03_大纲/INDEX.md`
- 更新 `06_追踪/伏笔追踪.md`
