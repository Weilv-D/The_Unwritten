# 文言写作辅助

基于《文心雕龙》等经典文论，辅助文言文写作与润色。

## 核心参考

- 研究资料：`08_灵感/研究资料/文心雕龙.txt`（刘勰《文心雕龙》全文）
- 写作指南：`08_灵感/研究资料/文言文写作指南（AI专用）.txt`（七卷完整方法论）
- 项目文风规范：`.claude/rules/core-style.md`

## 输入参数

用户输入：$ARGUMENTS（可以是：一段待润色的文本、一个写作需求描述、一个文体名称）

## 执行步骤

### 1. 判断任务类型

根据用户输入判断任务类型：

| 输入类型 | 执行路径 |
|---------|---------|
| 一段半文半白/白话文本 | → 文言润色（步骤2A） |
| 描述一个写作需求（如"写一段祭文""写一封书信"） | → 文体创作（步骤2B） |
| 一个问题（如"这个词文言怎么说""这个句式对不对"） | → 答疑指导（步骤2C） |
| 无具体输入 | → 进入交互模式（步骤2D） |

### 2A. 文言润色

对用户提供的文本进行文言化润色：

1. **诊断原文**：标注白话痕迹、现代语法渗透、虚词误用
2. **逐层改造**：
   - 词汇层：双音节→单音节压缩，古今异义替换，雅俗辨析
   - 句法层：去除"是"字判断句、修正语序、合理省略、宾语前置等特殊句式
   - 虚词层："之其者所""而以于""也矣乎哉"的精准运用
   - 篇章层：起承转合结构、气脉贯通、伏笔照应
3. **声律检查**：平仄相间、长短句错落、节奏控制
4. **输出**：润色后文本 + 改动说明（只说改了什么、为什么改，不列清单式总结）

### 2B. 文体创作

根据用户指定文体，按以下流程创作：

1. **确认文体**：论辨/序跋/赠序/书说/奏议/传状/碑志/杂记/辞赋/哀祭
2. **加载文体规范**：从写作指南中提取该文体的特征、结构模式、经典范例
3. **确认需求**：向用户确认——
   - 写作对象与场合
   - 期望风格（典雅/远奥/精约/壮丽等）
   - 关键内容要点
   - 字数范围
4. **创作**：严格遵循该文体的结构模式和语言规范
5. **自检**：
   - 文体特征是否吻合
   - 虚词运用是否精准
   - 用典是否自然贴切
   - 声律节奏是否协调
6. **输出**：正文 + 简要说明所用文体技法

### 2C. 答疑指导

针对文言写作的具体问题：

1. **词汇辨析**：提供古义、引申义、经典用例
2. **句式指导**：判断句/被动句/宾语前置/省略的适用条件
3. **虚词用法**："之其者所而以于也矣乎哉"等的语境辨析
4. **用典建议**：根据语义需求推荐典故，说明化用方式
5. **风格分析**：从"神理气味格律声色"八维度分析

### 2D. 交互模式

无具体输入时，进入引导模式：

向用户展示可选功能：
1. **润色模式**：粘贴一段文本，我来文言化
2. **创作模式**：告诉我文体和主题，我来创作
3. **学习模式**：问我任何文言写作问题
4. **诊断模式**：我分析一段古文的写法技法

## 核心原则

### 文言写作七要

1. **道先文后**：先立意，后措辞。"文原于道"，不可辞藻堆砌而无意
2. **气骨为要**：文章须有风骨——精神气质（风）与结构力度（骨）并重
3. **意在言外**：留白比填满更高明。"春秋笔法"，微言大义
4. **虚实相生**：实词为骨，虚词为魂。虚词调控语气、节奏、逻辑
5. **声律谐和**：平仄相间，长短错落，句读分明
6. **用典自然**：如盐入水，有味无痕。宁少毋滥，宁熟毋僻
7. **文道合一**：内容与形式统一，义理与辞章兼备

### 五大排雷禁区

1. **白话思维渗透**：禁用"是"字判断句、"的"字结构、"了着过"、"把/被"字直译
2. **第三人称代词作主语**：上古汉语"之""其"不作主语，用省略或重名
3. **古今异义误用**：每用一词先查古义（走=跑、去=离、涕=泪、妻子=妻+子）
4. **虚词滥用**："之""而""以""于"不可随意添加，须有语法依据
5. **词类活用无据**：使动/意动/名作状须有经典用例支撑，不可凭空杜撰

### 文体速查

| 文体 | 核心 | 结构模式 | 典范 |
|------|------|---------|------|
| 论辨 | 弥纶群言，研精一理 | 起（立论）→承（论证）→转（驳异）→合（升华） | 韩愈《原道》 |
| 序跋 | 提要钩玄，知人论世 | 介绍→评价→发挥→祝愿 | 欧阳修《梅圣俞诗集序》 |
| 赠序 | 赠人以言，情理兼备 | 追交情→写别情→勉祝愿 | 韩愈《送孟东野序》 |
| 书说 | 因事陈辞，因人设语 | 称谓→问候→叙事→议论→祝愿 | 司马迁《报任安书》 |
| 奏议 | 引经据典，陈情动人 | 引经典→陈现状→提建议→表忠心 | 诸葛亮《出师表》 |
| 传状 | 寓论断于叙事 | 世系→生平→功业→论赞 | 《史记》列传 |
| 碑志 | 铭诔尚实，不虚美 | 序（散文叙事）+铭（韵文赞颂） | 韩愈《柳子厚墓志铭》 |
| 杂记 | 随事立名，不拘一格 | 叙事→写景→议论→抒情 | 柳宗元《小石潭记》 |
| 辞赋 | 铺采摛文，体物写志 | 序→正文（铺陈+对偶）→乱/系 | 王勃《滕王阁序》 |
| 哀祭 | 情往会悲，文来引泣 | 追生平→抒哀情→评功业→寄哀思 | 韩愈《祭十二郎文》 |
