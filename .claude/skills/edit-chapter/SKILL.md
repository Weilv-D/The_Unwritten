# 修改章节

根据反馈或审阅报告，对已有章节进行针对性修改。

## 输入参数
章节编号：$ARGUMENTS（格式：卷1/ch003）

## 执行流程

### 1. 加载材料
- **必读规则**：`writing-style.md`、`forbidden-rules.md`
- `05_正文/$ARGUMENTS.md`（待修改的正文）
- `04_章纲/$ARGUMENTS.md`（对应章纲）
- 如有审阅报告：`06_追踪/审查报告/$ARGUMENTS_审查.md`

如果正文不存在，告知作者并停止。

### 2. 确认修改范围
**必须向作者确认**：
- 修改哪些部分？（特定段落/场景/对话/全章风格）
- 修改目标是什么？（审阅反馈/作者新想法/一致性修复）
- 是否需要保持原有字数？

### 3. 执行修改
- 针对性编辑，不改动未确认的部分
- 保持整体结构和风格一致
- 修改后的段落仍需符合风格规范

### 4. 简化自检
修改完成后执行 3 项检查：
- **字数**：修改后是否仍在 5500-6500 范围内
- **风格一致**：修改部分与原文风格是否统一
- **连贯性**：前后文衔接是否自然

### 5. 输出
- 展示修改内容（标注修改前/后对比）
- 简化自检结果
- 等待作者确认

### 6. 存档
作者确认后：
- 更新 `05_正文/$ARGUMENTS.md`
- 更新 `06_追踪/创作日志.md`

## 下一步
→ `/review-chapter`（确认修改质量）
