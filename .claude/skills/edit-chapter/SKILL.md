# 编辑章节

根据反馈或审查报告修改已有章节正文。

## 必读规则
- `core-style.md`（14条禁忌红线、文风锚定）

## 输入参数
章节编号：$ARGUMENTS（格式：卷1/ch003）

## 执行步骤

### 1. 加载素材
- `05_正文/$ARGUMENTS.md` 当前正文
- `04_章纲/$ARGUMENTS.md` 对应章纲
- `core-style.md` 风格规范
- 反馈/审查报告（如有）

### 2. 确认编辑范围

**与作者确认**：
- 修改类型（微调/段落级/场景级/结构性）
- 涉及哪些方面（文风/节奏/逻辑/角色/伏笔）
- 有无不可改动的关键段落

### 3. 执行修改
- 严格遵循风格规范
- 保持章纲锁定的"写什么"不变（如需改动请示作者）
- 修改后的段落与上下文自然衔接

### 4. 自检
按 `core-style.md` 的14条禁忌红线逐条检查修改部分：
- 每条标注 PASS/FAIL
- FAIL 项标注具体位置和修改建议

### 5. 输出
- 展示修改内容（标注修改位置）
- 自检报告
- 询问作者是否满意

### 6. 存档
作者确认后：
- 覆盖写入 `05_正文/$ARGUMENTS.md`
- 更新 `06_追踪/创作日志.md`
- 建议提交 Git
