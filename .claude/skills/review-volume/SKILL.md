# 审查卷次

对整卷进行四阶段综合审查。

## 必读规则
- `core-plot.md`（三幕推进法则、场景引擎）
- `core-style.md`（14条禁忌红线）

## 输入参数
卷次：$ARGUMENTS（如 卷1）

## 执行步骤

### 阶段1：结构检查
- 读取 `03_大纲/$ARGUMENTS_大纲.md` 和 `03_大纲/$ARGUMENTS_细纲.md`
- 检查三幕结构是否完整
- 检查十大序列节拍是否到位
- 检查伏笔的设置/推进/回收是否闭环
- 检查支线是否全部收束或有明确延续标记

### 阶段2：一致性检查
按 `/check-consistency` 的五维标准执行全卷一致性检查。

### 阶段3：质量审查
抽样4个关键章节（开场、转折、高潮、结局）按 `/review-chapter` 标准执行质量评审。

### 阶段4：输出综合报告

```
## 卷次审查报告
**卷次**：$ARGUMENTS

### 结构评估
- 三幕完整性：✅/⚠️/❌ [说明]
- 序列节拍：✅/⚠️/❌ [说明]
- 伏笔闭环：✅/⚠️/❌ [说明]
- 支线收束：✅/⚠️/❌ [说明]

### 一致性评估
[汇总一致性检查结果]

### 质量评估
[汇总抽样章节评审结果]

### 关键问题（必须修复）
1. ...
2. ...

### 优化建议
1. ...
2. ...

**综合评分**：X/10
```
