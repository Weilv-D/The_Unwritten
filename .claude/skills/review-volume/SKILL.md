# 审查卷次

## 必读

- `.claude/rules/core-plot.md`
- `.claude/rules/core-style.md`
- `06_追踪/审查报告/审查报告模板.md`

## 流程

1. **结构检查**
   - 读取 `03_大纲/卷次/$ARGUMENTS_大纲.md` 和 `03_大纲/卷次/$ARGUMENTS_细纲.md`。
   - 检查三幕、序列节拍、支线收束、伏笔闭环。

2. **一致性检查**
   - 按 `/check-consistency` 的五维标准执行。
   - 术语和能力边界必须回源文件查证。

3. **关键章节抽查**
   - 抽查开场、转折、高潮、结局四类章节。
   - 每个问题给出位置、触发规则、修改建议和需作者决策项。

4. **证据优先报告**
   - 将报告写入 `06_追踪/审查报告/卷X_卷次审查.md`。
   - 必须修复项在前，建议优化在后，综合评分只放末尾。

## 错误处理

- 缺卷大纲或细纲时停止，并建议先补对应规划。
- 正文未开始时只审查规划结构，不虚构章节质量。
