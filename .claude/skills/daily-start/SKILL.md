# 每日创作启动

为今日创作做准备，自动加载所需上下文并展示项目状态。

## 执行步骤

### 1. 读取项目状态
- 读取 `00_项目基础/进度追踪.md`
- 读取 `00_项目基础/任务清单.md`
- 读取 `06_追踪/创作日志.md`（最后5条记录）

### 2. 统计字数
扫描 `05_正文/` 目录下所有文件，计算总字数（中文字符数）。按卷次分别统计。

### 3. 汇报当前状态
向作者展示：

```
## 项目状态

### 整体进度
- 总字数：XXXX / 3,000,000（X.X%）
- 当前阶段：[世界观/人物/大纲/章纲/正文]
- 当前卷次：卷X

### 各卷进度
| 卷次 | 字数 | 章节 | 状态 |
|------|------|------|------|
| 卷1 | XXXX | X/100 | 进行中 |

### 最近活动
- [最近5条创作日志]

### 待办事项
- [从任务清单中提取高优先级项]
```

### 4. 确认今日目标
**必须询问作者**：
- 今天要做什么？（写作/审阅/设定/规划/灵感）
- 目标章节或任务
- 预计时长

### 5. 预加载上下文
根据今日目标自动加载对应材料：
- 写作：章纲 + 人物速查 + 上章结尾
- 审阅：待审正文 + 对应章纲
- 设定：相关现有设定文件
- 规划：大纲 + 已有设定
- 灵感：08_灵感/INDEX.md

### 6. 就绪确认
告知作者已加载的文件列表，准备开始工作。

### 7. 建议
根据当前进度，建议下一步操作。
