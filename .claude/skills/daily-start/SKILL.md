# 每日创作启动

为今日创作做准备，自动加载所需上下文。

## 执行步骤

### 1. 读取项目状态
- 读取 `00_项目基础/进度追踪.md`
- 读取 `00_项目基础/任务清单.md`
- 读取 `06_追踪/创作日志.md`（最后3条记录）

### 2. 汇报当前状态
向作者展示：
- 整体进度（已完成字数/总字数）
- 当前所处阶段（世界观/人物/大纲/章纲/正文）
- 待处理任务列表
- 上次会话摘要

### 3. 确认今日目标
**必须询问作者**：
- 今天要做什么？（写作/审阅/设定/规划/灵感）
- 目标章节或任务
- 预计时长

### 4. 预加载上下文
根据今日目标自动加载对应材料：
- 写作：章纲 + 人物速查 + 上章结尾
- 审阅：待审正文 + 对应章纲
- 设定：相关现有设定文件
- 规划：大纲 + 已有设定
- 灵感：08_灵感/INDEX.md

### 5. 就绪确认
告知作者已加载的文件列表，准备开始工作。
