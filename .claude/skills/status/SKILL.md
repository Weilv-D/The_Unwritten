# 项目状态

快速展示项目进度和待办事项。

## 执行步骤

### 1. 读取核心文件
- `00_项目基础/进度追踪.md`
- `00_项目基础/任务清单.md`
- `06_追踪/创作日志.md`（最后5条记录）

### 2. 统计字数
扫描 `05_正文/` 目录下所有文件，计算总字数（中文字符数）。
按卷次分别统计。

### 3. 输出摘要
向作者展示：

```
## 项目状态

### 整体进度
- 总字数：XXXX / 3,000,000（X.X%）
- 当前阶段：[世界观/人物/大纲/章纲/正文]
- 当前卷次：卷X

### 各卷进度
| 卷次 | 字数 | 章节 | 状态 |
|------|------|------|------|
| 卷1 | XXXX | X/100 | 进行中 |
| ... | ... | ... | ... |

### 最近活动
- [最近5条创作日志]

### 待办事项
- [从任务清单中提取高优先级项]
```

### 4. 建议
根据当前进度，建议下一步操作。

## 下一步
无固定后续。根据建议选择对应技能。
