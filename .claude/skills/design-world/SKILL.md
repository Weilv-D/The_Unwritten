# 设计世界观

按洋葱层展开法，渐进式设计或扩展世界观设定。

## 必读规则
- `core-world.md`（冰山涟漪策略、势力生态动力学、命名术语规范）

## 执行步骤

### 1. 确认设计范围
向作者展示当前 `01_世界观/INDEX.md` 的状态，确认要设计/扩展的领域。

### 2. 识别层级
判断本次设计属于哪一层：
- **第一环（个人）**：修行资源、功法、灵物
- **第二环（势力/地域）**：灵脉、领地、传承、秘境
- **第三环（天下格局）**：势力范围、贸易、跨州界关系
- **第四环（大道/天命）**：金性、业位、天地兴衰

### 3. 提供设计方案
提供 2-3 个设计方案，每个包含：
- 核心概念
- 与已有设定的关联点
- 潜在的戏剧冲突点
- 标注信息可见度（读者何时得知）

**等待作者选择。**

### 4. 渐进展开
选定的方案展开为完整设定，包含：
- 核心定义
- 关键规则/限制
- 与其他设定的交互关系
- 角色层面的影响

### 5. 一致性检查
检查新设定与已有世界观的一致性：
- 术语是否符合命名规范
- 是否与已有设定矛盾
- 层级关系是否合理

### 6. 输出
- 写入 `01_世界观/` 对应子目录
- 更新 `01_世界观/INDEX.md`
