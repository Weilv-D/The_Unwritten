# 初始化项目

幂等检查项目基础设施，创建所有缺失的文件。

## 执行步骤

### 1. 扫描缺失文件

检查以下类别是否存在：

**项目基础**：
- `00_项目基础/进度追踪.md`
- `00_项目基础/任务清单.md`
- `00_项目基础/统一方法论与工作流指导.md`

**世界观**：
- `01_世界观/INDEX.md`

**人物**：
- `02_人物/INDEX.md`
- `02_人物/人物速查.md`
- `02_人物/角色模板.md`

**大纲**：
- `03_大纲/INDEX.md`
- `03_大纲/故事梗概.md`

**章纲**：
- `04_章纲/章纲模板.md`

**追踪**：
- `06_追踪/创作日志.md`
- `06_追踪/伏笔追踪.md`
- `06_追踪/知识图谱.md`

**灵感**：
- `08_灵感/INDEX.md`
- `08_灵感/剧情灵感.md`
- `08_灵感/场景灵感.md`
- `08_灵感/对话碎片.md`
- `08_灵感/主题思考.md`
- `08_灵感/技法笔记.md`
- `08_灵感/待分类.md`

**目录**：
- `05_正文/` 目录（及其下按卷次子目录）

### 2. 报告状态
向作者展示缺失文件清单。

### 3. 等待确认
**等待作者确认后再创建。**

### 4. 创建缺失文件
按模板内容创建缺失文件（空模板或基础骨架）。

### 5. 验证与提交
- 确认所有文件已创建
- 建议提交 Git
