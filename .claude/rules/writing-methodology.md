# 创作方法论·总纲

> 核心原则：**让一切情节都有其"必然性"——源于历史因果、势力利益或人物性格，杜绝"为了让主角赢而赢"。**

---

技法按使用阶段拆分为三个文件，通过 `paths` 自动激活，避免无关内容占满上下文。

## 设定构建（methodology-world.md）

激活时机：编辑 `01_世界观/` 或 `02_人物/` 时

| 技法 | 一句话 |
|------|--------|
| 势力分层博弈法 | 高层博弈在低层的投射，战斗由此获得超越胜负的意义 |
| 因果宿命锚点 | 历史因果和血脉诅咒决定行为边界，制造"挣扎于宿命"的张力 |
| 道统人格化 | 力量体系与性格深度绑定，修行某种道统必然塑造某类特质 |
| 历史包袱驱动 | 祖先恩怨、誓言、荣耀是当下行为的重要动机 |

## 剧情规划（methodology-plot.md）

激活时机：编辑 `03_大纲/` 或 `04_章纲/` 时

| 技法 | 一句话 |
|------|--------|
| 对话即剧情 | 用密谈/谈判/争吵揭示世界观、推动情节、展现多方立场 |
| 关系网驱动变量 | 预设密集关系网，关键时刻让关系成为改变局势的变量 |
| 探索-战斗-收获循环 | 保证持续推进感，每次冲突后都有实质性收获 |
| 多线悬念编织 | 同时铺设多个时间尺度不同的悬念，连绵不绝的钩子 |
| 小剧情核心公式 | 目标 + 障碍 + 意外 + 转折，至少包含前两者和后两者之一 |
| 六种常见类型 | 谈判/试探/救援/背叛/信息揭露/情感冲突，各有设计要点 |
| 场景设计五步法 | 确定目标 → 设置要怕 → 开场状态 → 信息流动 → 对话动作 |
| 场景技巧与检查清单 | 打断偏离/两难选择/反向预期/多重目标/情绪升降机/道具妙用 + 五项检查 |

## 写作技法（methodology-writing.md）

激活时机：编辑 `05_正文/` 时

| 技法 | 一句话 |
|------|--------|
| 配角独立生命感五原则 | 利益自主/能力差异/独立社交/异步成长/有限视野 |
| 战斗三轴驱动 | 50%代价 + 30%智斗 + 20%意象，战斗不是证明强者而是展示代价 |
| 服务式视角切换 | 镜头追随掌握关键信息或处于行动中心的角色 |
| 利益-沉默双轴 | 60%利益先行 + 40%沉默表达，不对等关系为底色 |
| 信息揭露六原则 | 分层冰山/视角锚定/碎片拼图/延迟奖励/对话即揭露/空白留白 |
