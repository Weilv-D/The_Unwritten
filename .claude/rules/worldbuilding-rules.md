# 世界观构建规则

## 核心原则

### 渐进式揭示
- 世界观通过对话、行动、独白自然融入，严禁大段设定说明
- 读者认知与主角同步升级，信息分层释放
- 第一卷仅展示"第一环"（个人层面），后续逐层展开

### 力量体系规范
- 真君/仙人在前三卷只有侧面描写，高高在上
- 真人前期高不可攀，第二卷下场
- 真君第四卷下场，仙人第五卷下场
- 每个境界有明确的能力边界和限制

### 势力构建要求
每个势力必须定义：
- 核心利益（在争什么）
- 组织结构（怎么运作）
- 关键人物（谁说了算）
- 与其他势力的关系（盟/敌/暗线）
- 当前面临的内忧外患

### 地理与资源
- 灵脉分布决定势力格局
- 修炼资源（灵石、灵物、功法）是权力基础
- 秘境、遗迹是争夺焦点

### 命名规范
- 地名：意象+地理（玉京、青墟、玄渊）
- 势力名：意象+组织形式（紫霄宗、太初殿、天衡阁）
- 道法/神通：『』包裹，A+B+C结构
- 功法/丹药/宝物：【】包裹

## 一致性检查
- 新增设定必须与已有文件对照，避免矛盾
- 术语首次出现必须有上下文暗示其含义
- 跨文件引用的设定保持统一
