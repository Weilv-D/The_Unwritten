# 大纲设计规则

## 层级依赖

大纲必须严格按层级推进，不可跳级：
```
故事梗概 → 卷次大纲 → 卷次细纲 → 章纲 → 正文
```
- 上级大纲未完成时，不设计下一级
- 每级大纲完成后，更新对应的 INDEX.md

## 节拍器理论要求

每卷大纲必须覆盖十大序列：
`開場 → 切入 → 激勵 → 進展 → 轉折 → 二次轉折 → 危機 → 導入 → 高潮 → 結局`

每个序列必须填充八大要素：
`身份 | 欲望 | 动作 | 问题(内) | 阻力(外) | 结果 | 正主题 | 负主题`

## 伏笔规划

每级大纲必须包含伏笔规划：
- **粗纲**：定义本卷设置/回收/跨卷推进的伏笔清单
- **细纲**：每章标注伏笔操作（设置/推进/回收）
- **章纲**：明确本章伏笔操作的具体位置和方式

## 支线编排

支线必须与主干有明确的勾连：
- 定义切入章节、关键信息节点、与主干的交叉点、收束章节
- 支线目的必须明确（主题探讨/势力展示/情报提供）

## 章节标题规范

- 四字成语或古风意象风格
- 6-12字
- 暗示本章核心事件但不过度剧透
- 同卷内不可重复用字

## 一致性检查点

每级大纲完成后，必须对照：
- 世界观设定（01_世界观/）
- 人物档案（02_人物/）
- 前卷大纲（如有）
- 故事梗概（03_大纲/故事梗概.md）
