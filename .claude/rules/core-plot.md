# 核心剧情法则 (Core Plot)

> 以人物关系为引擎，宏观定势微观随变。
>
> **定位**：本文件承载剧情设计中的**硬规则、默认模板与参考场景类型**。大纲的分层理论和逆向拆解法见 `统一方法论与工作流指导.md`。

## 一、 大局统筹：定点锚定·三幕式

所有大纲（卷次）设计默认按以下层级逐级落地（大纲→细纲→章纲→正文）。其中比例与节拍是**默认模板**，可因卷规模、题材密度和高潮展开而偏离。

### 1. 战役层：三幕推进（默认模板）

任何一卷都可以先按以下三幕估算时间线：
1. **第一幕：局势与引线 (约 20%)**：各派势力粉墨登场，旧平衡由于一枚意料外的筹码（功法、异宝、死亡、秘境开启）被彻底打破，主角被迫卷入漩涡并确立初步的**核心诉求**。
2. **第二幕：激化与反复 (约 60%)**：冲突白热化，结盟与背叛的高发期。主角遭遇连续的阻扰，付出惨重代价（如伤残、丧友、失势），但也在血战与博奕中获取核心信息与关键助力。
3. **第三幕：压轴大收束与余波 (约 20%)**：大格局中的终极变量介入，所有铺垫引发核心高潮（决战或破局）。之后必须留有足够的篇幅进行**残局盘点、论功行赏与战利品分割**，并埋下直指下一卷的深远悬念。

### 2. 战略层：结网与倒推 (Global Level)
- 单卷仅设定**起止状态**与**核心大事件节点**。这由世界观或顶级势力的博弈所形成的大势决断，无法逆转。
- 这不是“先定结局再去扭曲角色”，而是**这些势力与主角基于其五维性格，不可避免地做出的选择必然导致的这几个终局节点**。

## 二、 战术推进：微观场景引擎 (Chapter Level)

推演每一章时，必须放弃“全知视角大甩卖”，彻底代入当前 POV 的**信息盲区**。

### 1. 场景推演公式

章纲场景至少应明确以下四项：

**场景 = P.O.V 目标 + 阻碍/信息差 + 意外变量 + 决断转折**

1. **目标 (Goal)**：他当前急迫需要什么？（破阵、延寿、套取情报、保全某人）
2. **阻碍 (Obstacle)**：谁/什么在阻止他？（环境险恶、功法克制、强敌暗伺、己方内杠的隐患）
3. **意外变量 (Twist)**：发生的一件出乎各方预料的变化（第三方乱入、本该是弱势方的绝地反击、情报被证实是假局）。
4. **决断转折 (Resolution)**：他因为这个变故不得不做出决定，引发了下一场的困局或者转机。

### 2. 对话即博弈（交涉模版）
- 严禁大段设定说明书。
- 所有对话都潜藏着刀光剑影。每一方的发话，都在计算着如何套取对方的底牌，或是释放烟雾弹。
- 采用**多方阵营交替审视法**：通过细微的动作（摩挲玉环、瞳孔微缩、语气的停顿），展现POV与NPC之间的心智交锋。

### 3. 多线悬念交错编织
- 同时铺设多个维度的钩子（终极谜团、中期阴门、当下生死危机）。一个悬念刚解开，其背后的代价或新牵扯出的大患立即接轨，保证“读者心中至少有一个急于寻找答案的痛点”。

## 三、参考场景类型

以下六类场景是常用模板，用于启发章纲设计，不作为强制 checklist：

1. **谈判/博弈局**：各方围绕筹码和底线展开试探。
2. **试探/摸底局**：信息在闲谈、旁敲、反问中交锋。
3. **杀运/斗法局**：重点展示代价、相克与战机，而非纯数值斗殴。
4. **背叛/反转局**：倒戈必须有利益或性格伏笔。
5. **遗迹/秘辛勘破局**：秘密要足够沉重，能改写后续战略。
6. **利益瓜分局**：战后分配是第二轮权力斗争，不是简单盘点。

## 四、 角色互文

### 1. 角色与主题的共鸣

每个核心角色都承载作品核心主题的一个侧面——对同一个命题给出不同的回答。他们的回答最终带来不同的结局，构成主题的完整表达。

### 2. 互文对子

角色之间形成互文——一个人的选择照亮另一个人的选择。设计角色时考虑他与其他角色的互文关系：相似之处是什么？在关键选择上的分歧是什么？分歧导致了什么不同的结局？

互文关系常见形态：镜像（同类人走向不同结局）、对照（不同类人面对相同困境）、反衬（一个接受命运，一个反抗命运）。
