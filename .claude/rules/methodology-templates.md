# 场景与剧情模板库

> 从场景类型到剧情模板，按需查阅的设计参考。每种模板都是"起承转合"的基本积木。

---

## 一、6种基础场景类型

| 类型 | 核心冲突 | 设计要点 |
|------|----------|----------|
| **谈判/博弈** | 两方或多方交换条件 | 每方必须有对方想要的东西，且各有底线。通过让步、威胁、欺骗推进 |
| **试探/摸底** | 一方隐藏真实意图，另一方试图看穿 | 对话要"话里有话"，通过表情、停顿、转移话题等细节暗示 |
| **救援/驰援** | 一方陷入危机，另一方赶来 | 制造"差一点来不及"的紧张感，救援者的到来要改变局势 |
| **背叛/反转** | 原本的盟友突然倒戈 | 需要提前埋下伏笔（利益冲突、性格缺陷），反转后要快速展现后果 |
| **信息揭露** | 角色得知某个秘密 | 秘密要足够"重"（改变认知、影响决策），揭露方式要自然 |
| **情感冲突** | 角色之间的情绪碰撞 | 冲突要源于具体事件而非空泛的情绪，结局可以是和解、加深裂痕或悬而未决 |

## 二、21种剧情模板

### 模板选择原则

1. **目标决定模板**：获取资源→入洞天/献宝求援/丹道炼制；化解危机→借势压人/谈判/识破离间；击败敌人→破局反杀/围点打援/诈降；提升地位→立威/会盟/联姻
2. **角色能力决定模板**：武力型→入洞天/破局反杀/围点打援；智谋型→识破离间/拉拢/借势压人；外交型→登门拜访/谈判/献宝求援；锋刃型→追杀逃亡/渡劫突破
3. **剧情阶段决定模板**：开局→立威/开辟洞府/登门拜访；发展→入洞天/联姻/收徒；高潮→会盟/围点打援/破局反杀；收尾→谈判/辞行/势力整合
4. **模板可组合**：一个剧情弧往往由多个模板嵌套而成

---

### 冲突对抗型（10种）

#### 入洞天

**【模板定义】** 进入与外界隔绝的特殊空间，限时限规则争夺机缘。

**【核心要素】** 空间规则（灵识受限、法力压制）、时间限制、多方势力、机缘分布、信息差

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 征兆显现 | 引发关注，各方聚集 |
| 入局准备 | 展示各方实力，埋设伏笔 |
| 进入空间 | 分散落点，建立初始格局 |
| 探索解谜 | 展示空间风貌，获取小收获 |
| 宝物出世 | 核心机缘显现，引发争夺 |
| 混战争夺 | 多方势力大混战 |
| 空间异变 | 增加压力，加速收束 |
| 脱离盘点 | 安全退出，清点收获 |

**【写作要点】**
- 前半程慢节奏铺设信息，后半程快节奏混战高潮
- 空间的特殊规则必须清晰且持续影响行动
- 至少三方势力参与，宝物价值值得各方争夺
- 主角收获有"意料之外"成分

---

#### 借势压人

**【模板定义】** 自身实力不足，借用更高位者的威名、信物或制造假象迫使对方退让。

**【核心要素】** 势差、信物/凭证、信息不对称、利益诱惑、时间窗口

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 困境建立 | 遭遇无法独立解决的困境 |
| 势的确认 | 明确可借用谁的势 |
| 接触目标 | 与借势对象建立联系 |
| 展示势能 | 亮出信物/信息，震慑对方 |
| 给予台阶 | 提供对方服从的理由和好处 |
| 对方权衡 | 描写对方心理斗争 |
| 达成妥协 | 对方屈服，目标达成 |

**【借势类型】** 借人名、借信物、借信息、借形势、借谣言

**【写作要点】**
- 势差要足够大
- 借势者要有"狐假虎威"的机敏
- 被压者要有"宁可信其有"的心理
- 读者知道是借势，担心被识破

---

#### 破局反杀

**【模板定义】** 绝境中利用信息差和隐藏底牌逆转局势。

**【核心要素】** 绝境感、信息差、底牌、时机、震慑效果

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 绝境铺设 | 主角陷入明显劣势 |
| 对手松懈 | 对手以为胜券在握 |
| 暗中蓄力 | 主角暗中准备底牌 |
| 时机到来 | 对手露出破绽 |
| 底牌爆发 | 连续释放底牌 |
| 局势逆转 | 对手重伤/败退 |
| 震慑全场 | 其他敌人不敢轻举妄动 |

**【底牌类型】** 隐藏实力、隐藏法器、隐藏法术、隐藏援军、隐藏规则

**【写作要点】**
- 绝境描写要足够压抑
- 底牌要有提前埋设的伏笔
- 反杀要"连续"而非"一击"：破防→定身→重创
- 反杀后要有"震慑"效果

---

#### 追杀逃亡

**【模板定义】** 遭遇无法抗衡的追杀者，通过速度比拼、地形利用、断尾求生逃生。

**【核心要素】** 追杀者实力碾压、时间/距离限制、逃亡者急智、牺牲/代价、转机/援军

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 追杀开始 | 追杀者出现，实力悬殊 |
| 全力逃遁 | 逃亡者全速逃离 |
| 第一次危机 | 追杀者逼近 |
| 急智应对 | 用计谋拉开距离 |
| 第二次危机 | 再次被追上 |
| 付出代价 | 为逃生付出代价 |
| 逃出生天 | 时限到或援军至 |

**【代价类型】** 法器损失、伤势、寿元、底牌消耗、同伴牺牲

**【写作要点】**
- 节奏呈"波浪式"：逃遁（平缓）→被追上（紧张）→拉开（平缓）→再被追上（更紧张）
- 每次"被追上"都比上次更危险，代价更大

---

#### 会盟

**【模板定义】** 多方势力为共同利益聚集，表面合作暗中博弈。

**【核心要素】** 共同利益/敌人、各方诉求、盟约仪式、暗中交易、盟约脆弱性

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 召集 | 召集方发出邀请 |
| 各方到场 | 展示势力格局 |
| 表面客套 | 寒暄中的试探 |
| 核心议题 | 提出共同目标 |
| 任务分配 | 各方认领任务 |
| 暗中博弈 | 私下交易和算计 |
| 盟约达成 | 形式上的统一 |
| 执行变数 | 盟约执行中出现问题 |

**【写作要点】**
- 明确每方利益诉求，埋下"盟约必被破坏"的伏笔
- 角色定位：召集者/主力/辅助/变数/观望者

---

#### 论道

**【模板定义】** 两个或多个角色就道统、理念展开对话，传递信息塑造人物。

**【核心要素】** 论道场合、话题切入点、观点分歧、信息交换、论道成果

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 场合设置 | 建立论道场景 |
| 话题引入 | 从具体问题切入 |
| 观点陈述 | 一方先陈述观点 |
| 深入探讨 | 双方交锋 |
| 信息揭示 | 透露关键信息 |
| 论道收束 | 总结或留下悬念 |

**【论道五功能】** 信息传递、人物塑造、关系变化、伏笔埋设、世界观构建

---

#### 渡劫突破

**【模板定义】** 角色冲击更高境界，经历内外考验。

**【核心要素】** 突破契机、突破准备、突破过程、天地异象、外力干扰、成败结果

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 契机出现 | 突破时机到来 |
| 闭关准备 | 资源盘点，护法安排 |
| 突破开始 | 冲击瓶颈 |
| 异象显现 | 天地异象引发关注 |
| 外力干扰 | 敌人趁机来袭 |
| 护法抵御 | 盟友抵御干扰 |
| 突破成败 | 最终结果 |
| 后续影响 | 对格局的改变 |

**【描写四层次】** 内视层（气海变化）、外显层（天地异象）、社会层（各方反应）、大道层（果位呼应）

---

#### 识破离间

**【模板定义】** 识破敌人挑拨离间的计谋，将计就计反制。

**【核心要素】** 离间计、信息不对称、识破过程、将计就计、反制成果

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 离间计实施 | 敌人开始挑拨 |
| 主角察觉 | 发现异样 |
| 验证推测 | 确认离间计存在 |
| 将计就计 | 设计反制方案 |
| 反间实施 | 让敌人的棋子反水 |
| 真相大白 | 离间计被揭穿 |
| 反制成果 | 离间者遭殃 |

---

#### 围点打援

**【模板定义】** 以围攻某目标为幌子，实际伏击前来增援的敌军主力。

**【核心要素】** 佯攻目标、真正目标、信息差、伏击地点、时机把握

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 选定佯攻目标 | 选择能调动敌军的目标 |
| 围而不攻 | 施加压力但不决战 |
| 调动敌军 | 让敌军主力被牵制 |
| 奇袭真正目标 | 主力绕道攻击真正目标 |
| 敌军慌乱 | 真正目标被袭，敌军动摇 |
| 伏击援军 | 截杀回援的敌军 |
| 佯攻变真攻 | 原目标顺势拿下 |

---

#### 诈降

**【模板定义】** 假意投降敌方，取得信任后关键时刻反水。

**【核心要素】** 投降动机、投名状、信任建立、关键位置、反水时机

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 走投无路 | 角色陷入绝境 |
| 假意投降 | 表示愿意归顺 |
| 献投名状 | 提供有价值的情报/行动 |
| 取得信任 | 敌方逐渐放松警惕 |
| 潜伏等待 | 等待反水时机 |
| 关键时刻反水 | 在敌方最致命时反戈 |
| 里应外合 | 与原来的盟友配合破敌 |

---

### 社交博弈型（8种）

#### 登门拜访

**【模板定义】** 主动前往对方领地，通过礼数和利益交换达成目的。

**【核心要素】** 拜访时机、礼数、礼物、真实意图、拜访结果

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 选择时机 | 确定拜访时间 |
| 备礼登门 | 准备礼物，正式登门 |
| 寒暄客套 | 表面客套中的试探 |
| 切入正题 | 委婉或直接提出诉求 |
| 对方回应 | 同意、拒绝、或提条件 |
| 深入谈判 | 就具体条款协商 |
| 达成协议 | 明面或暗中达成一致 |
| 告辞余韵 | 拜访结束，留下伏笔 |

**【写作要点】** 拜访背后的政治意图才是核心，对话要"话中有话"

---

#### 联姻

**【模板定义】** 两个势力通过婚姻巩固联盟交换利益。

**【核心要素】** 联姻动机、对象选择、聘礼/嫁妆、仪式、联姻效果

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 联姻提议 | 一方主动或第三方撮合 |
| 对象选择 | 确定联姻的具体人选 |
| 聘礼谈判 | 资源交换的讨价还价 |
| 定亲仪式 | 形式上的确认 |
| 联姻执行 | 实际成婚或相处 |
| 联盟效果 | 联姻带来的实际利益 |
| 潜在变数 | 埋设可能的变数伏笔 |

---

#### 献宝求援

**【模板定义】** 以宝物为代价向强者请求援助。

**【核心要素】** 危机（有时效性）、宝物（与求援难度匹配）、求援对象、人情/承诺、求援结果

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 危机降临 | 遭遇无法解决的危机 |
| 盘点资源 | 清点可动用的宝物和人脉 |
| 选择对象 | 确定向谁求援 |
| 献宝陈述 | 献上宝物，陈述危难 |
| 对方权衡 | 求援对象权衡利弊 |
| 应允/拒绝 | 对方做出决定 |
| 援军/结果 | 援助到来，危机解决或失败 |

---

#### 立威

**【模板定义】** 在新环境中碾压式击败挑衅者，震慑全场确立地位。

**【核心要素】** 新环境/新地位、挑衅者（有分量但注定被击败）、碾压式胜利、是否下杀手、震慑效果

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 背景建立 | 需要立威的新环境 |
| 挑衅者出现 | 有人不服或试探 |
| 冲突升级 | 从言语到行动 |
| 碾压胜利 | 压倒性击败 |
| 追加震慑 | 继续扩大战果 |
| 观者反应 | 全场震惊 |
| 地位确立 | 立威后的实际效果 |

---

#### 谈判

**【模板定义】** 两方或多方在筹码基础上博弈达成协议。

**【核心要素】** 谈判背景、筹码、底线、讨价还价、协议

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 谈判背景 | 双方都有谈判需求 |
| 设定场合 | 选择谈判地点 |
| 初次报价 | 双方开出初始条件 |
| 讨价还价 | 亮筹码、揭底牌 |
| 僵持突破 | 第三方调停或外部压力 |
| 达成协议 | 最终约定 |
| 协议执行 | 实际履行（可能有变数） |

---

#### 收徒

**【模板定义】** 发现可造之材，通过考验收为弟子。

**【核心要素】** 发现、考验、拜师仪式、传授、师徒关系发展

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 发现可造之材 | 偶然或刻意寻找 |
| 初步观察 | 考察徒弟资质 |
| 表明意愿 | 提出收徒 |
| 拜师仪式 | 正式确立关系 |
| 传授赐宝 | 传授功法或赐予法器 |
| 师徒相处 | 关系的日常发展 |
| 徒弟成长 | 徒弟为主角贡献力量 |

---

#### 拉拢

**【模板定义】** 通过利益诱惑和情感投资将中立者或敌方边缘人物争取到自己一方。

**【核心要素】** 拉拢对象（处境尴尬/怀才不遇/与原主有矛盾）、突破口、利益诱惑、情感投资、拉拢结果

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 识别对象 | 发现可拉拢之人 |
| 接触试探 | 建立联系，试探态度 |
| 找到突破口 | 发现对方的需求 |
| 抛出橄榄枝 | 许诺利益 |
| 对方权衡 | 描写对方心理斗争 |
| 成功/失败 | 对方答应或拒绝 |
| 后续验证 | 拉拢对象的实际贡献 |

---

#### 辞行/送别

**【模板定义】** 角色因远行/出征/闭关与亲友告别，展示关系、埋设伏笔。

**【核心要素】** 离别原因、离别场景、情感流露、交代后事、赠礼/信物

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 离别原因 | 为什么要离开 |
| 辞行场景 | 与谁告别、私下还是公开 |
| 情感流露 | 克制的情感表达 |
| 交代后事 | 托付未了之事 |
| 赠礼/信物 | 有象征意义的物品 |
| 离别余韵 | 离别后的思念或伏笔 |

---

### 经营建设型（3种）

#### 开辟洞府

**【模板定义】** 发现并占据灵地，布阵经营打造成修行基地。

**【核心要素】** 选址（灵脉/天险）、清理（驱赶原住民）、布阵、功能分区、经营发展

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 发现地点 | 如何发现这处宝地 |
| 清理占据 | 解决原住民或竞争者 |
| 布置阵法 | 建立防御体系 |
| 洞府探索 | 发现洞府中的秘密 |
| 功能建设 | 建立各功能区 |
| 安置人口 | 迁徙族人/弟子 |
| 洞府升级 | 持续投入，不断升级 |

---

#### 丹道/器道炼制

**【模板定义】** 收集材料开炉炼制获取珍贵丹药或法器。

**【核心要素】** 丹方/器图、材料收集、炼制过程、波折、成丹/器

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 获得丹方 | 如何得到配方 |
| 材料收集 | 获取所需材料 |
| 开炉准备 | 丹炉、火脉、辅助材料 |
| 炼制过程 | 手法、火候描写 |
| 意外波折 | 炼制中的问题 |
| 成丹时刻 | 丹药出炉 |
| 效果验证 | 丹药的实际使用 |

#### 势力整合

**【模板定义】** 通过武力、外交、联姻等手段将分散势力统一麾下。

**【核心要素】** 整合对象、整合手段（武力/外交/联姻）、阻力、利益分配、整合效果

**【标准结构】**

| 节拍 | 功能 |
|------|------|
| 整合契机 | 为什么现在整合 |
| 确定目标 | 整合哪些势力 |
| 武力征服 | 对抵抗者的军事行动 |
| 外交收服 | 对中立者的拉拢 |
| 利益分配 | 整合后的安排 |
| 制度建设 | 建立统治体系 |
| 整合效果 | 势力是否真正统一 |

## 三、模板组合技术

### 三种组合方式

**1. 主模板+辅助模板**
每个大剧情有一个主模板，辅以多个辅助模板。辅助模板在主模板的间隙中展开，丰富剧情层次。

**2. 模板嵌套**
一个模板内部可以嵌套另一个模板。例：谈判（外层）内嵌借势压人（内层）+ 识破离间（内层）+ 拉拢（内层）。嵌套使单一场景承担多重功能。

**3. 模板变奏**
同一模板在不同情境下有不同变奏——标准版（多方混战）、简化版（低烈度竞争）、升级版（更高格局的博弈）。变奏保持模板核心不变，调整规模和烈度。
