# 详细写作风格规范

> 本文件是 CLAUDE.md 中"风格速查"的完整版，写作时以本文件为准。

---

## 一、语言基调

**半文半白，古典简练。** 严禁现代口语、网络用语、轻松白话。

必用虚词：`遂` `亦` `业已` `已然` `遑论` `岂能` `何故` `何至于此` `未尝不` `不曾想` `俨然`
必用动词：`沉吟` `踱步` `颔首` `稽首` `躬身` `悚然而惊` `不置可否`
必用短语：`若有所思` `瞬息万变` `心惊肉跳` `威风凛凛` `不可一世` `面色惨白` `智珠在握`

行文必须书面化、古风化。多使用四字格成语（如"若有所思"、"瞬息万变"、"心惊肉跳"），避免任何现代网络词汇或口语化表达。

---

## 二、色彩材质系统

**色彩**：`玄`（黑） `青` `紫` `金` `白` `赤` `墨` `灰`
**材质/光效**：`玉` `琉璃` `玄纹` `宝光` `煞气` `氤氲` `华光` `神辉` `冷意` `杀机`

组合示例：
- `金光琉璃般的光华` / `紫玉般的光泽` / `玄黑煞气弥漫` / `青色氤氲缭绕`
- `他身着一袭玄纹黑云道袍，腰间悬着一枚紫金玉佩`

---

## 三、术语标记规则

- **『』** 包裹核心概念、道法、神通名称：`『晞炁』` `『赤断镞』` `『君蹈危』`
- **【】** 包裹具体功法、丹药、宝物：`【明彰日月】` `【回春九转诀】` `【焚天赤业刀】`

命名原则：
- 意象+概念结合（门/剑/河/山 + 天/道/阴阳/生死/君王）
- 适当用生僻字（晞/渌/宸/阙）
- A+B+C三层结构（焚天·赤业·刀）
- 化用经典概念（厥阴/太素/甲子），赋予故事中独特含义

---

## 四、『』内心独白规范（核心中的核心）

- **高频使用**，与外在言行形成**表里反差**
- 三层结构：`观察触发 → 快速分析/回忆比对 → 得出结论/制定策略`
- 严禁用独白直接陈述完整阴谋或替代剧情展示
- 独白是分析推演，揭示计算和恐惧，不是内心独白的剧情说明

示例：
> 他面上不动声色，右手却在袖中悄然掐了个诀。『此人气息内敛，步伐看似闲适，实则每一步都暗合阵位。紫金玉佩……内务府的信物。朝廷的人，来得比我预想中早了三日。』

---

## 五、[景-人-心] 段落公式

每个叙事段落遵循三段式递进：

```
[景] 1-3句环境/氛围描写，确立基调
[人] 人物登场，通过外貌/神态/动作/简短对话推进
[心] 情节触发后，用『』插入内心推演
```

完整示例：
```
[景] 天色将暮，西风裹挟着沙尘掠过城头，旌旗猎猎作响。

[人] 来人青衫磊落，腰悬一枚紫金玉佩，面色淡然，对守将的审视视若无睹。他在城门下驻足，微微仰头，唇角浮起一丝若有若无的笑意。

[心] 『此人气息内敛，步伐暗合阵位。紫金玉佩……内务府的信物。来得比我预想早了三日。』守将面上不动声色，右手却在袖中悄然掐了个诀。
```

## 七、称谓等级系统

- 真君/仙君 → 仙人 → 真人 → 修士 → 小修/散修
- 大人/前辈/道友/晚辈/属下
- 上位者：动作幅度小（`挑眉` `垂眸` `指尖轻敲`），语言简练
- 下位者：动作幅度大（`曲身行礼` `跪倒在地` `不敢直视`），必用敬称

对话必须严格体现身份等级。上位者语气简洁不容置疑，下位者态度恭谨必用敬称。

## 八、战斗描写公式

```
1. 起手：人物微动作 + 咒语/敕令（掐诀/并指/合十/抚须）
2. 显化：天地变色 + 具象异象生成（色彩/形状）
3. 碰撞：异象撞击 + 声音（轰隆/铿锵）
4. 后果：环境破坏 + 对手状态（面色一白/吐血/气息萎靡）
```

关键要求：
- 所有技能/招式用『』包裹，名称古风晦涩
- 施法前有微动作描写（掐诀/并指/合十/抚须/敕令）
- 法术具象化为宏大物体或自然现象
- 强制使用修仙界色彩（玄黑、紫金、青玉）和材质（琉璃、金石、汞水）
- 伤害描写强调对`道基` `性命` `气机` `太虚联系`的损伤，而非只写流血
- 动作描写极其简练、迅速、结果导向

## 九、叙事逻辑要求

### 利益驱动原则
所有角色的行为动机基于家族利益、道途进阶或生存博弈。没有无缘无故的爱恨。

### 侧写手法
禁止上帝视角旁白。通过角色观察、内心推测、暗示性对话间接勾勒事实。

### 镜头切换
- **宏观**：大场面、神通异象，语言华丽磅礴
- **微观**：人物博弈，焦点落在眼睛、面部、手部

### 特色句式模板
- `A也好，B也罢，终究……`
- `非是……而是……`
- `……已然到了……的地步。`

## 十、章节结构（6000字）

```
第一幕：钩子切入（20%，约1200字）
  - 3句话内建立场景、绑定视角、抛出核心悬念

第二幕：发展博弈（60%，约3600字）
  - 对话、行动、视角切换推进核心事件
  - 信息三层释放：表层 → 中层 → 深层
  - 人物状态实质性变化

第三幕：收束留白（20%，约1200字）
  - 阶段性结果 + 新伏笔 + 结尾句有余韵
```

## 十一、自检清单

### 创作前（5项）
- 已读取当前章纲
- 已读取视角人物速览
- 已读取上一章结尾
- 已确认本场基调和节拍位置
- 已确认出场人物和相关设定

### 创作中（每1000字，8项）
- 无直接情绪告知词
- 场景内视角固定
- 内心独白使用『』且非直白陈述
- 对话有信息推进
- 段落遵循[景-人-心]结构
- 术语使用『』或【】包裹
- 半文半白风格一致
- 信息密度足够

### 创作后
执行12条禁止规则逐条检查（见 forbidden-rules.md）
