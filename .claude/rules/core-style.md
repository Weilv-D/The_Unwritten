# 核心风格与禁令法典 (Core Style)

> 写作正文时的语言质感、视角边界与质量分层以此为准。
>
> **定位**：本文件承载语言层面的**硬规则、默认模板与深审项**。创作流程和检查清单见 `统一方法论与工作流指导.md`。

## 一、 语言基调与修辞

### 1. 文风锚定
- **半文半白，古典简练**：严禁现代口语（如"搞定"、"郁闷"、"对吧"）、网络用语和轻小说式的插科打诨。
- **参考词汇区**：
  - 可优先考虑的虚词：`遂` `亦` `业已` `已然` `遑论` `岂能` `何故` `未尝不` `俨然`
  - 可优先考虑的动词：`沉吟` `踱步` `颔首` `稽首` `躬身` `悚然而惊` `不置可否`
  - 可优先考虑的状态：`若有所思` `瞬息万变` `心惊肉跳` `威风凛凛` `不可一世` `面色惨白` `智珠在握`
- **词的分量感**：关键用词必须同时完成叙事和人物塑造。"乘势避害""定鼎之重""宁损实而不损名"这类词本身即在定义角色形象，不是装饰。
- **留白**：默契与共识用一句收束（如"黑与金的眼睛对视了，一切已尽在不言之中"），不解释双方在想什么。

### 2. 展示优先，必要时可直陈
- **默认倾向**：优先通过动作、场景、关系张力来承载情绪与信息。
- **允许直陈**：当需要压缩信息、切换节奏、交代必要结论时，可以简洁直陈，但不能滑向说明书式旁白。
- **动作传情**：如"解剑→捏住剑腰→擦拭剑刃"可传达杀意；这是常用技法，不是唯一合法写法。

## 二、 视角纪律与信息屏障

### 1. 严格的限知视角
- **禁止上帝视角全览**：在绝大多数场景下，镜头必须死死贴在**当前POV（视点人物）**的脑后。他看不到的背后暗杀不能写全景，他不知道的敌方谋划不能用旁白强行解释给读者听。
- **多方对峙的交替内聚焦**：仅在各方大佬谈判论道的关键群像戏中，允许镜头在段落间自然切换到不同角色的微表情和心理独白，展现出强烈的"言行不一"和信息差。
- **史官式视角切换**：允许在宏观（多线并行的天下大势）与微观（个体感官沉浸）之间自由切换，但每次切换必须有明确的叙事目的。关键转折时以极短句切换视角（如"让他破"），瞬间重置叙事节奏。
- **内外结合**：通过他者视角观察角色（外部），再通过角色自身言行和回忆揭示内心（内部），两者交替塑造立体人物。

### 2. 世界观嵌入式叙事
- **禁止说明书**：绝不允许出现大段的设定科普或"正如你所知"的降智对话。力量体系、千年恩怨必须通过法宝的缺损、战利品的分配、谈判时的隐晦痛脚来"侧面折射"。
- **对话即博弈**：人物对话必须充满潜台词和交锋感，不仅仅是信息传递，更是性格和智谋的碰撞。对话中的信息是武器——角色用信息试探、说服、威胁、引诱。

## 三、FAIL 级硬规则

以下问题一旦出现，应视为当前版本**不合格**，优先修复：

1. **术语与设定冲突**：生造白话功法、术语前后矛盾、与源文件规则冲突。
2. **POV 越界**：旁白替角色知道他不该知道的信息，或无叙事目的地滑入全知视角。
3. **说明书式设定倾倒**：旁白停下剧情，大段讲设定、历史、体系。
4. **角色或势力为推剧情降智**：关键选择违背既有立场、欲望、资源逻辑。
5. **明显现代杂糅**：古风段落里出现明显现代口语、网络语或现代社科套话。
6. **关键冲突被私自消解**：发现章纲、设定、时间线、伏笔冲突却不告知作者而擅自圆过去。

## 四、重点改进项

以下项目是重要质量指标，但不直接作为废稿线：

- 情感泛滥与说教
- 句式拖沓
- 对话注水
- 儿戏战斗
- 势力空壳
- 节奏平铺
- 直白内心戏过多
- 情绪单一
- 公式化初稿痕迹明显

## 五、深审项（终稿 / 精修阶段）

> 本节用于判定“是否够好”，不要求在首稿阶段全部完成。

### 1. 自评禁令
- 严禁对自己生成的正文使用"无瑕""上佳""无可挑剔"等绝对肯定评价
- 自检报告必须至少标注 1 处可改进点，即使红线全部 PASS
- 自评维度使用"达标"/"可接受"/"待改进"三档，不使用"优秀"/"完美"

### 2. 古文词汇三重校验
- 实词校验：每个文言实词须能在先秦两汉典籍、唐宋古文、或《玄鉴仙族》原文中找到用例。不确定的词标注"待查证"
- 虚词校验：虚词使用须有语法依据，参照 wenyan-writing skill 的五大排雷禁区
- 名词校验：生造复合名词必须标注为"新造"，不可伪装为既有古语

### 3. 可读性底线
- 每段完成后用白话复述核心信息。复述不出的段落标记为"可读性不足"
- 需要注释才能理解表面含义（非深层含义）的段落不合格

### 敏感内容的藏笔处理

触碰暴力、权谋、情欲等敏感内容时，采用"藏"的手法：

- **暴力**：写后果与环境影响，不写过程。"拎着断裂长枪走出" + "大殿如豆腐被炸成雷霆土石"——通过人物状态和环境变化侧面烘托。
- **权谋**：用天道运行的隐喻替代直白争斗。"用蜀帝的气象来填明阳"将残酷博弈抽象为规则运作，既点明本质，又保持玄妙距离感。
- **情欲**：走主观感受侧面烘托，不直接描写。"忽而觉得面前好像是一片金黄"通过感受传达特质，保持文字干净。
