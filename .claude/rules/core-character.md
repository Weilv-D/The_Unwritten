# 核心角色法则 (Core Character)

> 配角有独立生命，行为由核心欲望驱动；坚决抵制任何为剧情服务的"工具人"现象。
>
> **定位**：本文件承载角色设计的**底层逻辑与具体法则**。角色的**创作流程**（六步法：位置→欲望→范式→弧光→关系→细节）见 `统一方法论与工作流指导.md` 第三节。

---

## 一、角色的结构生成

### 1. 位置先于性格

角色不是"是什么人"，而是"处于什么结构"。性格、行为、选择，首先由他在权力/资源/血缘结构中的**位置**决定，其次才由天性调节。

设计角色时，先画"位置图"——他在家族/宗门/势力中处于什么位置？这个位置赋予他什么资源？什么限制？什么必须完成的使命？

位置决定行为逻辑：长房长子有继承焦虑，旁系子弟有攀附需求，外姓门客有证明自己的冲动，散修有审时度势的本能。同一件事，处于不同位置的人会有截然不同的反应。

### 2. 生态位

每个角色在故事生态中占据一个生态位——位置决定了生存策略。

| 生态位 | 特征 | 生存策略 |
|--------|------|----------|
| 天命者 | 命数加身，大势所趋 | 顺应天命，借势而起 |
| 守成者 | 继承而非开创，压力来自内部 | 稳扎稳打，平衡各方 |
| 谋士型 | 自身实力有限，依附强者 | 智谋输出，证明价值 |
| 孤臣型 | 被主流排斥，独立生存 | 隐忍自保，等待时机 |
| 边缘人 | 出身或血统被歧视 | 超额付出换取认可 |
| 降将 | 旧主已失，新主未信 | 立投名状，证明忠诚 |
| 中间人 | 处于两方势力之间 | 左右逢源，或被迫站队 |

确定生态位后，行为自然生成——他必须做什么来维持这个位置？面临的最大威胁是什么？可能的上升路径和退路是什么？

## 二、五维核心定义与自主性

### 1. 五维核心定义

每个出场的具名角色必须预设：
1. **表层身份**：社会角色（世家弟子/散修联盟/宗门特使等）。
2. **隐藏身份**：他真正效忠的暗线、秘密（如有）。
3. **核心欲望**：驱动一切的核心动机（求生/复族/执念/对某种大道的探寻）。
4. **致命弱点**：性格缺陷、执念盲区或软肋。
5. **关键关系节点**：他与场上其他3-5个人的利益/情感引线。

### 2. 自主性三铁律

- **利益绝对自主**：配角拥有独立于主角的价值排序。生死关头，他们会基于自身利益/宗门利益做出果断的自保、切割甚至倒戈，不可被简单斥为"背叛"或"降智反噬"。
- **能力领域壁垒**：配角应在某一方面强于主角且具有不可替代性，让他在某段主线享有专属高光，而非"主角的移动技能包"。
- **独立时间线**：当镜头不在他身上时，他仍在为自己的目标谋划、结交、争夺，当再次登场时，他的势力与境界必然已经悄然发生变局。

每个配角在他自己的故事里都是主角——他有自己的欲望、关系网络、信息盲区，他的选择从他的视角看是合理的。

## 三、欲望与恐惧

### 1. 三层欲望模型

欲望分三层，层与层之间存在张力甚至矛盾。三层欲望的博弈，是角色内在动力的核心结构。

**【第一层：生存欲望】**
- 内容：活下去、保护亲人、维持基本尊严
- 驱动强度：极强（生死关头）
- 特点：被动的、反应性的

**【第二层：位置欲望】**
- 内容：提升或巩固自己在结构中的位置
- 驱动强度：持续性的中等强度
- 特点：主动的、竞争性的

**【第三层：超越欲望】**
- 内容：超越现有结构，追求更高意义（道途、长生、复仇、理想）
- 驱动强度：间歇性爆发，一旦启动不可逆转
- 特点：可能违背前两层欲望

**关键**：三层欲望之间的矛盾是角色内在张力的来源。什么情况下他会为了超越欲望牺牲位置？什么情况下他会为了生存压制超越欲望？矛盾爆发的那一刻，就是角色最闪耀的时刻。

### 2. 恐惧：欲望的负空间

欲望的反面是恐惧。完整的心理画像必须包含他**最害怕什么**。

- **生存恐惧**：怕死、怕失去至亲
- **丧失恐惧**：怕失去位置、身份、能力
- **意义恐惧**：怕一切无意义、怕被遗忘、怕辜负

用整个故事去逼迫他面对这个恐惧。角色在恐惧面前的选择，定义了他是什么样的人。

### 3. 讨喜范式

避免完美人设，赋予角色悲剧感或复杂感：
- **残缺守护者**：为了守护某物，付出了不可逆转的代价（如目盲、断途、折寿）。
- **负重凡人**：资质平庸，却被逼扛起宗门或家族大运，走得步步泣血。
- **清醒的野心家**：不掩饰对权力的渴望，但也绝对遵守自己定下的底线与契约。
- **温柔强者**：拥有碾压级实力却选择克制与守护，反差制造张力。
- **被误解者**：行为看似反派，实则有深层的合理动机，揭示时反转认知。

## 四、弧光与代价

### 1. 两种成长路径

**蜕变型**：角色经历事件后，价值观、行为模式发生根本改变。
- 遭遇无法用旧模式应对的危机 → 旧信念被证伪 → 找到新信念 → 在新模式中经受考验

**固化型**：角色经历事件后，核心特质不但没有改变，反而被强化和确认。
- 遭遇考验核心信念的事件 → 在压力下坚持原有信念 → 付出代价但证明了自己的选择 → 从此更加坚定

两种路径是对命运的不同回答，没有优劣之分。

### 2. 代价法则

**角色的每一次成长，都必须支付相应的代价。没有代价的成长是虚假的。**

代价的类型：
1. **关系代价**：失去重要的人
2. **道德代价**：做违背原则的事
3. **能力代价**：失去某种能力或可能性
4. **认同代价**：被误解、被孤立
5. **生命代价**：以寿命或生命为代价

代价越沉重，成长越真实。

## 五、关系动力学

### 1. 电荷与势差

每个人物关系都带有"电荷"——正电荷（吸引、亲近、亏欠）、负电荷（排斥、仇恨、竞争）。

任何关系中都存在"势差"——一方比另一方拥有更多权力、资源、信息或情感资本。势差的类型：实力势差、位份势差、信息势差、情感势差。

**势差的动态变化**是戏剧张力的核心来源。势差缩小、拉大、甚至逆转——逆转的时刻往往是最具戏剧性的时刻。

关系的四维度：
- 血缘：天然的、难以摆脱的强电荷
- 利益：可变的、算计性的中等电荷
- 道义：平时弱电荷，但爆发时极强
- 情感：非理性的、可能违背利益的电荷

### 2. 关系契约

每一段关系都有隐形的"契约"——双方默认的权利义务。当契约被破坏时，关系发生质变。

为每段核心关系写一句"契约条款"，然后设计事件让一方或双方"违约"。违约的时刻，就是关系戏剧性爆发的时刻。

## 六、人性极灰与情感克星

### 1. 极灰环境下的角色张力

没有纯粹的善恶，只有**极端的生存压力带来的策略选择**。
- 反派在自己立场的解释下，一定是极其合理、合逻辑甚至合乎"大义"的。
- 正派在某些时刻，为了大局必然会做出极其冷酷和残忍的战略舍弃。
- **代价守恒**：角色的每一次越阶、每一次破局，都必须在肉体、寿命、情感或气运上留下不可磨灭的沉痛代价。

### 2. 情感描写的极致克制

- **不倾诉、不滥情**：修仙界的森严等级与残酷生态，注定任何感情（亲情、友情、爱情）的表露都是奢侈且危险的。
- **行动替代**：最深的羁绊是通过生死关头的"优先救助"、"资源分配的悄然倾斜"、"随口的承诺在关键时刻兑现"来展示的，绝不使用直白的心理描写大呼"多么感动/悲痛"。
- **残垣断壁**：一段深刻的关系往往伴随着不可言说的误解、立场的撕裂。直到其中一方陨落，才将那份压抑彻底定格。

## 七、道统与命运

### 1. 道统作为命运参数

道统不仅是功法，更是一种命运的可能性集合。它赋予角色：
1. **能力边界**：能做什么，不能做什么
2. **性格倾向**：力量反噬人格（修雷法者易烈易折，修厥阴者易绝情诡谲）
3. **社交网络**：同道统者天然亲近或竞争
4. **命运路径**：道统的历史和宿敌决定了可能遭遇什么

设计角色时，把道统作为"命运参数"输入：这个道统在历史上发生过什么？这个道统的人通常有什么结局？角色如何在道统的框架内走出自己的路？

### 2. 命数与选择

不可选择的是出身、血脉、天资；可选择但有限制的是道统、立场、阵营；完全自由的是关键时刻的抉择。

命数不是枷锁，而是角色展现自由意志的舞台。读者想看的，不是"命数应验了"，而是"在命数之下，他做出了这样的选择"。

### 3. 角色的完成

一个角色的故事，在他回答了自己的**核心命题**时抵达终点：

- **身份命题**：我是谁？
- **道途命题**：我为何修行？
- **关系命题**：我与某人的关系如何解决？
- **价值命题**：什么值得我付出生命？

完成的标志：他回答了（或永远无法回答）自己的核心命题，为此付出了不可逆的代价，读者感到"这个人物的故事讲完了"。
