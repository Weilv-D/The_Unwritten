# 第023章 · 别问

## 元信息

- **卷次**：第一卷
- **序列定位**：激励
- **POV视角**：江既白
- **预计字数**：6000字
- **基调**：日常

---

## 八大要素

| 要素 | 内容 |
|------|------|
| 身份 | 新弟子，正在摸索师门的规矩和边界 |
| 欲望 | 了解更多关于师父和这个师门的信息。但规矩是"别问" |
| 动作 | 与凌青鸢在井边打水时对话，试探性地了解师门的运行方式 |
| 问题 | "别问"的规矩让他不适——前世的逻辑是"信息越多判断越准"，这里的逻辑是"不该知道的不问" |
| 阻力 | 凌青鸢的话里有双层含义——告诉他规矩，同时在提醒自己。她的防线比他的问题更深 |
| 结果 | 得到了几条关键信息。但每条信息都带来了新的疑问 |
| 正主题 | 凌青鸢的提醒是真心的——虽然包装在冷硬的措辞里 |
| 负主题 | "师父收人从来不解释"——这句话本身就说明了陆无咎的师徒关系建立在信息不对称上。师父知道弟子不知道的事，而弟子被要求不追问 |

## 三幕结构（6000字分配）

### 第一幕：钩子切入（20%，约1200字）

- **开场基调句**：井边的辘轳吱呀作响。凌青鸢在打水，江既白来洗碗。两个人在井台边站了一会儿，谁都没先开口。
- **核心悬念**：凌青鸢和江既白的距离在拉近还是保持？她的主动指点是破冰还是偶然？
- **场景**：客院·井台边，午后。日常、朴素、两人之间的沉默比话长。

### 第二幕：发展博弈（60%，约3600字）

- **核心事件**：凌青鸢与江既白第一次正式对话。关于师父、关于修行、关于不追问的规矩。
- **信息释放**：
  - 表层：两人在客院井边打水时碰上。凌青鸢主动说了几句话。核心是一句："师父收人从来不解释。别问为什么收你，也别问为什么不收别人。"
  - 中层：凌青鸢的话里有双层含义——表层是告诉江既白规矩，深层是在提醒自己：你也不该问。她不知道师父为什么收这个旁支帮手，但她选择不问。不问是弟子对师父的信任——或者服从
  - 深层：江既白从她的语气里听出了一种微妙的疲倦。不是对他的敌意——是对"被蒙在鼓里"的忍耐。她也在等答案。只是等的方式比他沉默得多
- **人物状态变化**：从"被动接受规矩"到"理解了凌青鸢也在信息黑洞中"——两人的处境有相似之处
- **关系变化**：师姐-师弟关系从"立规矩"推进到"坦诚交流"——虽然坦白的内容是"我们都不该问"

### 第三幕：收束留白（20%，约1200字）

- **阶段性结果**：凌青鸢临走时说了一句："师父给你的灵石，省着用。他带的不多。"——这句话暴露了她也知道师父的灵石不该只有这个水平
- **新伏笔**："他带的不多"——凌青鸢也在观察师父的异常。她知道的比她说的多，但她选择不多说
- **结尾句**：以凌青鸢的背影和那句话收束，暗示"师姐也不是什么都知道，但她选择不说"
