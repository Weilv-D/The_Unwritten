# 变更日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

格式说明：
- `feat`: 新增内容（章节、人物、设定等）
- `fix`: 修复问题（逻辑漏洞、时间线矛盾等）
- `docs`: 文档更新（设定修改、模板更新等）
- `refactor`: 内容重构（章节拆分、结构调整等）
- `review`: 审查结果（一致性检查、质量评估等）
- `style`: 格式调整（不影响内容的修改）
